# 研究报告

本目录存放基于 `experiments/results` 中已生成结果的人工研究报告。

约束：

- 只能引用已有协议、原始结果、分析结果、报告和对比文件。
- 不得把未扫描到的区域写成结论。
- 不得把趋势写成证明。
- 每个结论都应能回溯到 `analysis.json`、`summary.md`、`report.md` 或比较产物。
