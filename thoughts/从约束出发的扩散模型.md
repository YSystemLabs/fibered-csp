# 从约束出发的扩散模型

---

## 第一步：明确你在提议什么

传统扩散模型的核心是一个**学到的去噪网络** $\epsilon_\theta(x_t, t)$，它通过海量数据训练来逼近真实的反向转移核 $q(x_{t-1}|x_t, x_0)$。

你提议的是：**跳过训练，直接用解析的能量函数/约束函数来驱动反向过程。**

即：不学 $\epsilon_\theta$，而是直接构造每一步的去噪操作，方法是在每个噪声尺度上施加已知的约束，通过温度、惩罚、截断等手段控制约束的刚性。

用纤维化 CSP 的语言：

````text
传统扩散模型:  学习 f! (推前函子)，使截面相容
你的提议:      直接构造 f!，从已知的约束能量出发
````

---

## 第二步：为什么这个想法值得认真对待

### 2.1 传统扩散模型的"浪费"

训练 $\epsilon_\theta$ 实际上在做什么？它在学习数据分布 $p_{\text{data}}(x)$ 的**所有层级的结构**——从全局布局到像素级纹理。但如果你已经知道目标的结构约束（比如你知道要生成的图像必须满足某些物理规律、几何约束、语义规则），那么训练网络去"重新发现"这些约束就是浪费。

### 2.2 已有的先例

这个方向不是凭空而来，已有若干相关工作：

- **朗之万动力学（Langevin dynamics）**：给定能量函数 $E(x)$，直接用 $x_{k+1} = x_k - \eta \nabla E(x_k) + \sqrt{2\eta}\, \xi_k$ 采样。不需要训练。
- **模拟退火（Simulated Annealing）**：用温度调度 $T(t)$ 控制从粗到细的探索。
- **约束满足的信念传播（Belief Propagation）**：直接从约束的因子图结构出发，不需要学习。
- **扩散模型的无训练引导（training-free guidance）**：如 DPS (Diffusion Posterior Sampling)，在已训练的扩散模型上叠加额外的已知约束梯度。

你的提议比这些都更激进：**连基础的扩散模型都不训练，完全从约束出发。**

## 第三步：纤维化框架下的精确构造

### 3.1 给定数据

你拥有的不是训练数据，而是：

**(E1)** 一个层级范畴 $\mathcal{L} = \{L \to L{-}1 \to \cdots \to 1 \to 0\}$（$L$ 个层级，从粗到细）

**(E2)** 每层一个能量函数 $E_l : \mathcal{X} \to [0, \infty]$，其中 $\mathcal{X}$ 是全局构型空间

**(E3)** 三类控制参数：
- 温度调度 $T_l$：控制第 $l$ 层约束的"模糊度"
- 惩罚系数 $\rho_l$：控制约束的相对强度
- 截断阈值 $\tau_l$：硬性排除不可接受的构型

### 3.2 纤维的构造

每根纤维不再是抽象的值域，而是一个**具体的玻尔兹曼分布**：

$$\mathcal{V}_l = \text{由 } \pi_l(x) \propto \exp\Big(-\frac{\rho_l \cdot E_l(x)}{T_l}\Big) \cdot \mathbf{1}[E_l(x) \leq \tau_l] \text{ 定义的分布族}$$

三个参数各自的角色：

````text
温度 T_l:
  T_l 大 → π_l 平坦，约束松弛，允许广泛探索
  T_l 小 → π_l 尖锐，约束紧缩，集中在能量极小附近
  T_l → 0 → π_l 退化为 δ-函数，硬约束

惩罚 ρ_l:
  ρ_l 大 → 该层约束在全局能量中权重大
  ρ_l 小 → 该层约束可以被其他层"覆盖"
  ρ_l = 0 → 该层约束完全关闭

截断 τ_l:
  τ_l < ∞ → 硬边界，能量超过 τ_l 的构型被绝对禁止
  τ_l = ∞ → 无截断，纯软约束
````

### 3.3 推前函子 $f_!$ 的显式构造

对 $f : l \to l'$（从粗层到细层），推前函子不再是"学到的去噪网络"，而是一个**解析的条件分布**。

**构造方案**：给定粗层的分布 $\pi_l$，定义细层的"约束传播分布"为：

$$f_!(\pi_l)(x) \propto \pi_l(x) \cdot \exp\Big(-\frac{\rho_{l'} \cdot E_{l'}(x)}{T_{l'}}\Big) \cdot \mathbf{1}[E_{l'}(x) \leq \tau_{l'}]$$

即：在粗层分布的基础上，**叠加细层的约束**。

这等价于贝叶斯更新：

$$\pi_{l'}(x) \propto \pi_l(x) \cdot \text{likelihood}_{l'}(x)$$

其中 $\text{likelihood}_{l'}(x) = \exp(-\rho_{l'} E_{l'}(x) / T_{l'}) \cdot \mathbf{1}[E_{l'}(x) \leq \tau_{l'}]$。

### 3.4 重新索引 $f^*$ 的显式构造

$f^* : \mathcal{V}_{l'} \to \mathcal{V}_l$（从细层到粗层），对应于**边缘化掉细粒度信息**：

如果细层约束涉及的变量比粗层更多（细层有更多自由度），则 $f^*$ 是对额外自由度的积分：

$$f^*(\pi_{l'})(x_{\text{粗}}) = \int \pi_{l'}(x_{\text{粗}}, x_{\text{细}}) \, dx_{\text{细}}$$

但如果粗细层涉及相同的变量（只是约束的"粒度"不同），则 $f^*$ 是通过模糊化/平滑化实现的：

$$f^*(\pi_{l'})(x) = \int K_\sigma(x, x') \, \pi_{l'}(x') \, dx'$$

其中 $K_\sigma$ 是一个平滑核，$\sigma$ 由温度差 $T_l - T_{l'}$ 控制。

### 3.5 生成算法

**算法：纤维化约束采样（Fibred Constraint Sampling）**

````text
输入: 能量函数 {E_l}，温度调度 {T_l}，惩罚 {ρ_l}，截断 {τ_l}
输出: 样本 x ∈ X

1. 初始化: 从最粗层的先验采样
   x ~ π_L(x) ∝ exp(-ρ_L · E_L(x) / T_L) · 𝟏[E_L(x) ≤ τ_L]

2. 逐层细化: for l = L-1, ..., 0:
   a. 计算细层的条件能量:
      Ẽ_l(x) = ρ_l · E_l(x) / T_l
   
   b. 在当前 x 附近，按条件分布采样:
      x' ~ π(x'|x) ∝ exp(-Ẽ_l(x')) · K_l(x', x)
      其中 K_l 是以 x 为中心、宽度由 T_l 控制的转移核
   
   c. 截断检查:
      if E_l(x') > τ_l: 拒绝，返回步骤 2b 重新采样
   
   d. 更新: x ← x'

3. 返回 x
````

这个算法的每一步都有明确的纤维化对应：

````text
步骤 1:    基础层截面的初始化
步骤 2b:   推前 f! 的具体执行（条件采样）
步骤 2c:   截断 = 截面坍缩的检测（E > τ → ⊥）
步骤 2d:   截面的逐层延拓
````

## 第四步：三种控制手段的深层分析

### 4.1 温度 $T_l$：连续的约束刚性控制

温度控制的物理直觉很清楚：高温→约束松→广泛探索，低温→约束紧→精确优化。

但在多层级设定中，温度调度的设计变得微妙。考虑两种极端策略：

**策略 A：均匀退火**

$$T_l = T_{\max} \cdot \Big(1 - \frac{l}{L}\Big) + T_{\min} \cdot \frac{l}{L}$$

从粗层到细层线性降温。

**策略 B：阶梯式退火**

$$T_l = \begin{cases} T_{\text{高}} & l \geq l_{\text{临界}} \\ T_{\text{低}} & l < l_{\text{临界}} \end{cases}$$

在某个临界层级突然降温。

纤维化框架对此的分析：

策略 A 使得每步 $f_!$ 的温度差很小，每步的约束增量也小。这意味着 $f_!$ 在每步都"近似幺半"——局部可分解。但全程的累积效应可以是非幺半的。

策略 B 在临界层级产生一个大的温度跳变，对应的 $f_!$ 强烈非幺半——在这一步，大量约束同时激活，产生强涌现。

**预测**：策略 B 在临界层级容易产生"相变"——要么成功涌现出正确结构，要么灾难性地坍缩到错误结构。策略 A 更平滑但可能更慢。

**最优策略**可能是自适应的：用 $f_!$ 的幺半偏差 $\|\delta\|$ 在线监控涌现强度，当 $\|\delta\|$ 过大时减缓降温（给系统更多时间"协商"层间一致性）。

### 4.2 惩罚 $\rho_l$：约束的相对权重

惩罚系数与温度不同。温度控制"模糊度"（所有约束同时松弛），惩罚控制"优先级"（不同约束区别对待）。

在纤维化框架中，$\rho_l$ 改变的是**纤维内的格结构**：

$$V_l^{(\rho)} = \Big([0, \infty],\ +_\rho,\ \inf\Big), \qquad a +_\rho b = \rho \cdot a + \rho \cdot b$$

不同 $\rho_l$ 的层级在全局评分中贡献不同的权重。这对应于 Pareto 前沿的不同权重化——即在 Pareto 前沿上选择不同的点。

**实际用途**：当你知道某些约束比其他约束更重要时（如物理约束 > 美学约束），用 $\rho_l$ 编码这种优先级。

### 4.3 截断 $\tau_l$：硬边界

截断是最激进的控制手段。它把软约束变成硬约束：$E_l(x) > \tau_l$ 的构型被**绝对禁止**。

在纤维化框架中，截断直接对应于**坍缩**：

$$E_l(x) > \tau_l \implies \text{Score}_l(x) = \bot_l \implies \text{截面坍缩} \implies \text{所有更细层级退化}$$

截断的力量在于：它**保证**不会生成违反硬约束的样本。温度和惩罚只能在概率意义上压制违反，截断则绝对排除。

**风险**：如果截断过紧（$\tau_l$ 太小），可行域可能为空——截面不存在。算法会在步骤 2c 无限拒绝。

**诊断**：如果某层的拒绝率过高，说明截断阈值与上一层传播下来的分布不匹配。需要放松 $\tau_l$ 或增大 $T_{l-1}$（让上一层的输出更分散，覆盖更多可行域）。

### 4.4 三种手段的组合策略

三种手段各有所长：

````text
温度:  控制全局探索/开发平衡，平滑控制
惩罚:  控制层级间的相对重要性，精细调权
截断:  硬性保证不可违反的约束，但可能导致可行域为空
````

组合策略：

$$\pi_l(x) \propto \exp\Big(-\frac{\rho_l \cdot E_l(x)}{T_l}\Big) \cdot \mathbf{1}[E_l(x) \leq \tau_l]$$

外层用截断保证硬约束，中间用惩罚设定优先级，内层用温度控制探索程度。

在纤维化 CSP 的语言中：

````text
截断 τ_l → 定义纤维 V_l 的"可达区域"（非 ⊥ 的部分）
惩罚 ρ_l → 定义纤维间 ⊗ 的相对权重
温度 T_l → 定义纤维内的"度量尺度"（分布的集中度）
````

## 第五步：具体案例验证

### 案例：蛋白质构象生成

**目标**：从已知的物理约束生成蛋白质三维构象。

**能量层级**：

````text
L = 3 层:

l = 2 (最粗): 化学键约束
  E_2(x) = Σ (键长 - 标准值)² + Σ (键角 - 标准值)²
  硬约束，不可违反
  T_2 = 0.1,  ρ_2 = 100,  τ_2 = 0.01

l = 1 (中间): 二级结构约束
  E_1(x) = α-螺旋/β-折叠的氢键能量 + Ramachandran 角惩罚
  软约束，有统计偏好
  T_1 = 1.0,  ρ_1 = 10,  τ_1 = ∞

l = 0 (最细): 侧链相互作用
  E_0(x) = van der Waals + 静电 + 溶剂化
  极弱约束，高度自由
  T_0 = 2.0,  ρ_0 = 1,  τ_0 = ∞
````

**算法执行**：

````text
步骤 1: 从化学键约束出发，采样骨架构象
        π_2(x) ∝ exp(-100 · E_2(x) / 0.1) · 𝟏[E_2(x) ≤ 0.01]
        → 几乎确定地满足键长键角约束的骨架

步骤 2: 叠加二级结构约束，条件采样
        π_1(x) ∝ π_2(x) · exp(-10 · E_1(x) / 1.0)
        → 在满足化学键的前提下，形成 α-螺旋/β-折叠
        截面相容性: 二级结构不破坏化学键（坍缩检查）

步骤 3: 叠加侧链约束，条件采样
        π_0(x) ∝ π_1(x) · exp(-1 · E_0(x) / 2.0)
        → 在满足化学键和二级结构的前提下，优化侧链取向
        涌现: 大量侧链的集体效应产生三级结构
````

**$f_!$ 的非幺半性在这里的体现**：

二级结构约束 $E_1$ 对骨架的效应：

- 单个氢键的推前是局部的（$f_!$ 在单个约束上是幺半的）
- 但多个氢键的**协同推前**产生α-螺旋——这是多个弱氢键的集体效应，$f_!(\sum E_i) \neq \sum f_!(E_i)$

**这就是蛋白质二级结构的涌现。**

无需训练就能捕获它，因为物理约束是已知的。

## 第六步：可行性的严肃评估

### 6.1 核心困难：采样效率

算法的每一步都需要从条件分布采样：

$$x' \sim \pi(x'|x) \propto \exp(-\tilde E_l(x')) \cdot K_l(x', x)$$

当 $\tilde E_l$ 复杂（多模态、高维）时，这个采样可能极其困难。这正是朗之万动力学/MCMC 方法的老问题——在高维复杂能量面上混合（mixing）很慢。

**扩散模型之所以训练一个网络，恰恰是为了绕过这个采样困难。** 训练好的 $\epsilon_\theta$ 直接给出了反向转移核的近似，不需要做 MCMC。

所以你的提议面临的核心挑战是：

> **如何在不训练网络的情况下，高效地从每层的条件分布中采样？**

### 6.2 温度、惩罚、截断如何帮助

**温度的贡献**：高温使分布更平坦，降低能量面的"崎岖度"，加速混合。多层级温度调度让每步的采样更容易——你不是一步到位从复杂的目标分布采样，而是逐步从简单到复杂。

这实际上正是**模拟退火的逻辑**。已知模拟退火在某些条件下（对数降温）保证收敛到全局最优。

**截断的贡献**：截断直接缩小搜索空间。如果 $\tau_l$ 定义的可行域是凸的（或近凸的），在可行域内的采样比在全空间采样高效得多。

**惩罚的贡献**：不同的 $\rho_l$ 使每步聚焦于最重要的约束。低 $\rho$ 的约束在早期被忽略，减少能量面的复杂度。

### 6.3 什么情况下可行

**条件 A：能量函数具有"分层可分解性"**

如果每层的能量函数只涉及一小部分变量（或变量的低维投影），则每步的采样是低维问题。

````text
E_粗(x) 只依赖于 x 的低频分量（前几个 PCA 模式）
E_中(x) 只依赖于 x 的中频分量
E_细(x) 只依赖于 x 的高频分量
````

在这种情况下，每步的能量面都是低维的，采样高效。

**这在物理系统中常常成立**——不同尺度的约束自然地耦合到不同的自由度子空间。

**条件 B：能量函数的梯度可解析计算**

如果 $\nabla_x E_l(x)$ 可以高效计算，则可以用朗之万动力学在每步做梯度采样：

$$x' = x - \eta \nabla \tilde E_l(x) + \sqrt{2\eta T_l}\, \xi$$

这比无梯度的 MCMC 快得多。

**在很多物理/工程系统中成立**——能量函数是显式的物理公式，梯度可以解析或自动微分计算。

**条件 C：截断定义的可行域足够"温和"**

如果截断条件 $E_l(x) \leq \tau_l$ 定义的集合是凸集或星形集，则截断后的采样（受限朗之万/投影梯度）仍然高效。

### 6.4 什么情况下不可行

**情况 X：能量面高度非凸、多模态**

如果某层的能量函数有指数多个局部极小值，且局部极小值之间由高能量壁垒分隔，则在任何温度下采样都很慢——要么高温到看不到任何结构（$T \gg$ 壁垒高度），要么低温到被困在单个极小值中。

**这正是蛋白质折叠的根本困难。** 能量面（Bryngelson-Wolynes 漏斗景观）虽然有全局漏斗结构，但局部极小值极多。

对这种情况，纤维化约束采样**不能替代训练**。训练的网络之所以有效，是因为它从数据中学到了能量面的"捷径"——直接跳过壁垒。纯粹的 MCMC/朗之万无论怎么做温度调度都做不到这一点。

**情况 Y：能量函数无法解析表达**

例如"图像看起来像一只猫"——这个约束的能量函数无法写出解析表达式。你必须从数据中学习它。对这类问题，纤维化约束采样不适用。

### 6.5 适用域的划分

````text
适用:
  ✓ 物理系统（力场已知、梯度可计算）
  ✓ 工程约束（几何约束、运动学约束是解析的）
  ✓ 组合优化（CSP 的约束是显式的）
  ✓ 程序合成（类型约束、语法约束是形式化的）

部分适用:
  ~ 分子生成（力场已知但能量面复杂）
  ~ 材料设计（物理约束已知但搜索空间巨大）

不适用:
  ✗ 自然图像生成（"看起来真实"无法解析表达）
  ✗ 自然语言生成（语义连贯性无法解析表达）
  ✗ 任何需要从数据学习约束的任务
````

## 第七步：改进算法——混合策略

纯粹的纤维化约束采样和纯粹的训练扩散模型之间存在一个**连续谱**。最实际的方案可能是混合的。

### 7.1 已知约束直接施加，未知约束训练学习

````text
粗层: 已知的硬约束（物理定律、几何规则）
       → 直接用 τ 截断 + ρ 惩罚

中间层: 部分已知的软约束
       → 已知部分用 T 控制的玻尔兹曼分布
       → 未知部分用小型网络 ε_θ 学习

细层: 完全未知的统计约束
       → 传统扩散模型训练
````

纤维化框架自然地支持这种混合：粗层的 $f_!$ 是解析构造的，细层的 $f_!$ 是学到的，它们通过纤维化的截面相容性统一。

### 7.2 用约束引导替代训练

即使对需要训练的层，约束知识也可以**减少训练需求**：

- 截断减小搜索空间 → 需要更少的数据来覆盖
- 温度控制提供更好的初始化 → 需要更少的训练步数
- 惩罚编码先验知识 → 正则化效果，防止过拟合

**极端情况**：如果已知约束足够强（几乎完全确定了解空间），则只需要极少量训练来学习剩余的"自由度"。这本质上是**约束降维** + **在低维空间上的轻量级生成模型**。

### 7.3 分层采样的具体算法改进

**改进 1：层内使用哈密顿蒙特卡洛（HMC）**

HMC 利用能量函数的梯度信息，比普通 MCMC 高效得多。在每层的条件采样步骤中：

$$\text{HMC step: } (x, p) \mapsto (x', p') \text{ via Hamiltonian } H = \tilde E_l(x) + \frac{1}{2}|p|^2$$

利用辛积分器（leapfrog）保持能量近似守恒，接受率高。

**改进 2：层间使用退火重要性采样（AIS）**

从粗层分布过渡到细层分布时，不用单步采样，而用一系列中间分布做桥接：

$$\pi_l = \pi_{l,0} \to \pi_{l,1} \to \cdots \to \pi_{l,K} = \pi_{l-1}$$

其中 $\pi_{l,k}$ 是 $\pi_l$ 和 $\pi_{l-1}$ 的几何插值。AIS 提供层间转移的无偏估计，且可以计算归一化常数的比值。

**改进 3：截断后的反射采样**

当截断条件 $E_l(x) \leq \tau_l$ 定义的可行域有明确边界时，使用**反射 HMC** — 粒子碰到边界时弹回，保持在可行域内。

## 第八步：与重整化群的精确联系

这个构造与物理学的重整化群有一个非常精确的对应。

### 8.1 Wilson 重整化群

在量子场论/统计力学中，Wilson 重整化群做的事情是：

1. 从包含所有尺度的完整理论出发
2. "积分掉"最高频率（最细尺度）的自由度
3. 得到一个只包含低频自由度的有效理论
4. 重复，逐步粗粒化

这正是纤维化 CSP 中 $f^*$（重新索引/拉回）的具体物理实现：**从细层到粗层，积分掉精细自由度。**

反过来，纤维化约束采样做的是**反向重整化群**：

1. 从只包含粗尺度约束的理论出发
2. 逐步"恢复"细尺度自由度
3. 在每步用已知的物理约束引导细化
4. 最终得到包含所有尺度的完整构型

这恰好是 $f_!$（推前）的物理对应。

### 8.2 关键洞察

重整化群中有一个著名的现象：**积分掉快模式后，慢模式之间产生新的有效相互作用。** 例子：

- 积分掉电子的高能虚贡献 → 夸克之间产生跑动耦合
- 积分掉晶格的短波声子 → 长波声子之间产生非谐效应

这正是 $f_!$ 非幺半性的物理内容：**粗粒化/细粒化过程产生了新的交互，这些交互不是原始约束的简单叠加。**

纤维化约束采样的温度调度 $T_l$ 对应于重整化群中的**尺度参数（renormalization scale）$\mu$**。从高 $T$（粗尺度）到低 $T$（细尺度）的过渡，就是沿重整化群流的运动。

## 第九步：总结评估

### 可行性结论

| 方面 | 评估 |
|------|------|
| 理论一致性 | **高**。纤维化框架为这种算法提供了严格的数学基础 |
| 适用范围 | **中等**。适用于约束可解析表达的系统，不适用于需要从数据学习约束的系统 |
| 计算效率 | **取决于能量面性质**。对分层可分解的能量函数高效，对高度非凸的能量面可能很慢 |
| 相对于训练扩散模型的优势 | 不需要数据、不需要训练、物理约束精确满足、可解释 |
| 相对于训练扩散模型的劣势 | 无法处理隐式约束、采样效率可能低 |

### 最有前景的应用方向

1. **物理系统的构象采样**（蛋白质、材料、分子）——约束已知、梯度可计算
2. **工程设计优化**（结构设计、电路布局）——约束是显式的几何/物理规则
3. **程序合成/类型化生成**——约束是形式系统的推导规则
4. **混合方案：约束引导的轻量级扩散模型**——用已知约束减少训练需求

### 核心贡献

这个想法的实质贡献不是"又一个采样算法"，而是**一种新的设计原则**：

> **生成模型不必是端到端训练的黑盒。如果你知道系统的约束结构，你可以用纤维化 CSP 的框架直接构造生成过程——把每层的约束显式地编码为纤维化的结构，用温度/惩罚/截断控制层间的信息传递。**

这个原则适用的范围比纯粹的无训练采样更广。即使在需要训练的情况下，纤维化结构也提供了一种**分解训练任务**的策略——不训练一个大模型来学习所有层级，而是按层级分治，每层只学习该层的未知约束。
