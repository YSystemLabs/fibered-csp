# 纤维化富集范畴视角下的分层约束满足问题

---

## 摘要

本文提出一种基于纤维化富集范畴的约束满足问题（CSP）形式化框架。经典 CSP 及其变体（WCSP、模糊 CSP、概率 CSP）将所有约束置于同一个值域中评估，无法结构性地表达现实系统中普遍存在的**多层级约束**现象——不同层级的约束具有质性差异，层级之间存在前提性依赖与涌现效应。我们以 Grothendieck 双纤维化为核心工具，构造了一个统一框架，使得坍缩与涌现从纤维化的内在几何中自然浮现，而非作为外部公理引入。进一步地，我们提出并分析了一个关于计算复杂度的猜想：推前函子 $f_!$ 的幺半偏差是层间复杂度不可分解性的**关键不变量**——它是检测层间交互效应的必要工具，并与层内多态性判据、约束图宽度等条件构成互补的正交维度。我们给出了可分解方向的严格证明与不可分解方向的初步归约策略。

---

## 1 引言：多层级约束的普遍性

### 1.1 从图像谈起

考察一幅数字图像的结构。像素的排列服从多个层级的约束：

- **全局拓扑层**：像素必须按行列顺序排列，违反则图像退化为噪声。
- **区域关联层**：相邻像素团之间应保持空间连贯性，局部置换导致退化但整体可辨识。
- **单像素自由层**：单个像素的 RGB 取值几乎不受其他像素约束。

这三个层级之间的关系不是程度差异，而是**质性差异**：拓扑层崩溃后，区域层约束连"存在"的前提都没有了；大量像素层约束的集体效应可以涌现出区域层的结构。

### 1.2 一般模式

这种多层级约束结构并非图像独有：

| 领域 | 强约束（结构层） | 中约束（关联层） | 弱约束（自由层） |
|------|----------------|----------------|----------------|
| 自然语言 | 语法结构 | 句间连贯性 | 单词选择 |
| 排课系统 | 教室不冲突 | 课程分布合理 | 具体时段偏好 |
| 蛋白质折叠 | 化学键约束 | 二级结构 | 侧链旋转 |
| 城市规划 | 道路拓扑 | 功能区划分 | 建筑样式 |

> **注**：上表以三层为例是为说明之便。实际系统中层级数量和层级间拓扑可以更复杂——例如蛋白质折叠可细分为五层以上（共价键 → 骨架二面角 → 二级结构 → 结构域 → 四级结构），且层级之间未必构成线性链，可能是更一般的偏序结构。

共同的模式是：

> **有意义的结构 = 多层级约束以不同强度、不同逻辑共同塑形的结果。**

强约束定义合法状态空间，中约束引导偏好区域，弱约束留出多样性。

### 1.3 经典 CSP 的局限

经典 CSP 及其富集变体（VCSP）将所有约束映射到同一个值域 $V$，通过单一的 $\otimes$ 组合、单一的 $\bigoplus$ 聚合。无论 $V$ 多复杂，所有约束最终在同一个尺度上被比较。这导致三个根本性缺失：

1. **无法表达前提性依赖**：强约束的崩溃应使弱约束"无意义"，而非仅仅使全局评分变差。
2. **无法表达层级的内在逻辑差异**：拓扑层是布尔的，区域层是度量的，像素层是连续的——它们不应共享同一个代数。
3. **无法表达涌现**：大量弱约束的集体效应产生质变，单一的 $\otimes$ 不能编码"当约束足够多时跳到更高层级"。

## 2 经典 $\mathcal{V}$-富集 CSP 的回顾

为使后续发展自包含，先回顾标准的 $\mathcal{V}$-富集 CSP 定义。

### 2.1 富集基底

取一个完备幺半格 $\mathcal{V} = (V, \leq, \otimes, \bigoplus, \top, \bot)$，其中 $\otimes$ 用于组合约束贡献，$\bigoplus$ 用于聚合候选解的评分。典型实例：

- 硬 CSP：$V = \mathbf{2}$，$\otimes = \wedge$，$\bigoplus = \vee$
- WCSP：$V = [0, \infty]^{op}$，$\otimes = +$，$\bigoplus = \inf$
- 概率 CSP：$V = [0,1]$，$\otimes = \times$，$\bigoplus = \sup$

### 2.2 变量与赋值预层

变量集合 $X = \{x_1, \dots, x_n\}$ 生成作用域范畴 $\mathcal{P}(X)$（有限子集的包含偏序）。每个变量配备域 $D_x$，赋值预层为：

$$A : \mathcal{P}(X)^{op} \to \mathbf{Set}, \qquad A(S) = \prod_{x \in S} D_x$$

配备限制映射 $\rho_{T,S} : A(T) \to A(S)$（$S \subseteq T$）。

### 2.3 约束与求解

每条约束为二元组 $(S_\alpha, \varphi_\alpha)$，其中 $\varphi_\alpha : A(S_\alpha) \to V$。全局赋值 $a \in A(X)$ 的评分为：

$$\mathrm{Score}(a) = \bigotimes_{\alpha \in I} \varphi_\alpha(\rho_{X, S_\alpha}(a))$$

最优值为 $\mathrm{Opt} = \bigoplus_{a \in A(X)} \mathrm{Score}(a)$。

### 2.4 局限性诊断

所有约束 $\varphi_\alpha$ 映射到**同一个 $V$**，通过同一个 $\otimes$ 组合。这是一个扁平框架——无层级、无质性差异、无涌现。

## 3 纤维化 CSP 的定义

### 3.1 核心思想

用一个 Grothendieck 双纤维化 $p : \mathcal{V} \to \mathcal{L}$ 替代单一的值域 $V$。不同层级的约束映射到不同的纤维 $\mathcal{V}_l$，层间关系由纤维化结构内在地编码：

- **坍缩**从重新索引函子 $f^*$ 与截面化闭包自然推出
- **涌现**从推前函子 $f_!$ 的非幺半性自然推出
- 上一版定义中的**手写公理大部分被纤维化的内在性质所吸收**；仅余少量结构性条件（$f^*$ 保持 $\bot$ 和 $\bigwedge$、$f^*$ 的 oplax 幺半性）作为纤维化上的温和约束保留

### 3.2 完整定义

**定义 3.1**（层级范畴）。一个有限偏序集 $\mathcal{L}$（视为薄范畴），称为**层级范畴**。对象为约束层级，态射 $f : l \to l'$ 表示 $l$ 比 $l'$ 更精细（层级更低）。偏序集的反对称性自动排除循环的层级依赖。

> **约定**：态射方向"精细 → 粗糙"与 Grothendieck 纤维化的标准约定一致——推前函子 $f_!$ 沿态射方向从精细层推到粗糙层，重新索引函子 $f^*$ 逆态射方向从粗糙层拉回到精细层。
>
> **术语约定：偏序中的 oplax/lax 方向**。在偏序范畴中，"箭头方向"等同于"不等号方向"，容易引起混淆。本文统一如下：设 $F : (V, \leq) \to (W, \leq)$ 是保序映射，$\otimes_V, \otimes_W$ 分别为两侧的幺半运算。(1) $F$ 为 **oplax 幺半**：$F(u \otimes_V v) \geq_W F(u) \otimes_W F(v)$——"像的积不超过积的像"。(2) $F$ 为 **lax 幺半**：$F(u \otimes_V v) \leq_W F(u) \otimes_W F(v)$——"像的积不低于积的像"。(3) $F$ 为**强幺半**：两个方向同时成立（等式）。在代价语义 $[0,\infty]^{op}$ 中 $\leq$ 是反自然序，"$\geq_W$"对应自然序意义上的"$\leq$"，读者需留意此处的翻转。后文不再重复此约定。

**定义 3.2**（值域纤维化）。一个双纤维化 $p : \mathcal{V} \to \mathcal{L}$，满足：

**(V1)** 每根纤维 $\mathcal{V}_l = p^{-1}(l)$ 是一个完备幺半格 $(V_l, \leq_l, \otimes_l, \top_l, \bot_l)$。

**(V2)** 对每条态射 $f : l \to l'$（$l$ 精细，$l'$ 粗糙），重新索引函子 $f^* : \mathcal{V}_{l'} \to \mathcal{V}_l$：
  - 保持底元素：$f^*(\bot_{l'}) = \bot_l$（保证坍缩沿 $f^*$ 从粗糙层向精细层传播）
  - 保持所有下确界（$\bigwedge$）：$f^*(\bigwedge_{l'} S) = \bigwedge_l \{f^*(u) : u \in S\}$，对所有 $S \subseteq V_{l'}$
  - 是 oplax 幺半函子：$f^*(v \otimes_{l'} w) \geq_l f^*(v) \otimes_l f^*(w)$（含强幺半等式 $=$ 为特例）

> **关于 (V2) 三个条件的关系**：(i) 保持 $\bot$ 不被保持 $\bigwedge$ 自动蕴含——$f^*(\bigwedge V_{l'}) = \bigwedge \mathrm{Im}(f^*)$，一般不等于 $\bot_l$，因此需单独要求。(ii) 当 $\otimes_l = \bigwedge_l$（幺半运算恰为格的交运算，如 $\mathbf{2}$ 中 $\otimes = \wedge$）时，oplax 幺半由保 $\bigwedge$ 自动成立；但一般情况下 $\otimes$ 和 $\bigwedge$ 是不同运算（如 $[0,\infty]^{op}$ 中 $\otimes = +$ 而 $\bigwedge = \sup$），三个条件相互独立。(iii) oplax 幺半放宽了旧版本要求的强幺半条件（$=$），使更多自然的 $f^*$ 成为合法选择（如 §6.2 的 $e^{-\sqrt{c/255}}$），同时保留了命题 4.4 所需的结构。

**(V2')** （可选强化条件）$f^*$ 反映底元素：$f^*(v) = \bot_l \implies v = \bot_{l'}$。此条件保证精细层的坍缩能向上传播至粗糙层，适用于"精细层全面崩溃蕴含粗糙层结构无意义"的场景。并非所有应用都需要此条件。

**(V3)** 对每条态射 $f : l \to l'$，推前函子 $f_! : \mathcal{V}_l \to \mathcal{V}_{l'}$ 是 $f^*$ 的左伴随：$f_! \dashv f^*$。由于每根纤维是完备格而 $f^*$ 保持所有下确界（条件 V2），$f_!$ 的存在性由伴随函子定理（偏序集版本：完备格间保所有下确界的映射必有左伴随）**自动保证**——$f_!(v) = \bigwedge_{l'} \{u \in V_{l'} : v \leq_l f^*(u)\}$。因此 (V3) 是 (V1)+(V2) 的严格推论；将其显式列出是为了强调 $f_!$ 在后续构造中的角色。

**(V4)** $f_!$ **不要求**是幺半函子。

不同层级拥有不同的代数结构，例如：

$$\mathcal{V}_{\text{拓扑}} = (\mathbf{2}, \wedge, \vee), \qquad \mathcal{V}_{\text{区域}} = ([0,\infty]^{op}, +, \inf), \qquad \mathcal{V}_{\text{像素}} = ([0,1], \times, \sup)$$

> **关于框架轻重的说明**：由于每根纤维 $\mathcal{V}_l$ 是完备幺半格（特别是偏序集/薄范畴），纤维间的伴随 $f_! \dashv f^*$ 实质上是 **Galois 连接**。在此特化下，Grothendieck 纤维化的一般机器退化为"偏序集上的参数化 Galois 连接族"。本文选择保留纤维化语言有两个理由：(1) 截面范畴、lax 自然变换等纤维化固有概念使坍缩和涌现的表述简洁统一；(2) 为推广到非薄纤维（约束值域含非平凡态射）保留路径。读者可将本文所有构造等价地理解为参数化 Galois 连接族上的操作（详见附录 A.8）。

**命题 3.2'**（Grothendieck 构造）。值域纤维化 $p : \mathcal{V} \to \mathcal{L}$ **等价于**一个伪函子（pseudofunctor）$\Phi : \mathcal{L}^{op} \to \mathbf{CMLat}$（从层级范畴的对偶到完备幺半格的范畴），通过 Grothendieck 构造 $\mathcal{V} = \int_\mathcal{L} \Phi$ 给出。显式地：

- **对象**：$\mathrm{Ob}(\mathcal{V}) = \coprod_{l \in \mathcal{L}} V_l$（所有纤维的不交并）。对象 $(l, v)$ 由层级 $l$ 和纤维值 $v \in V_l$ 共同确定。
- **态射**：$(l, v) \to (l', u)$ 存在 $\iff$ $\mathcal{L}$ 中存在 $f : l \to l'$ 且 $v \leq_l f^*(u)$（等价地，$f_!(v) \leq_{l'} u$）。此处 $f^* = \Phi(f)$ 是伪函子的作用。
- **组合**：设 $(l, v) \xrightarrow{f} (l', u) \xrightarrow{g} (l'', w)$，即 $v \leq f^*(u)$ 且 $u \leq g^*(w)$。由 $f^*$ 保序得 $f^*(u) \leq f^*(g^*(w)) = (gf)^*(w)$，故 $v \leq (gf)^*(w)$，组合态射 $(l, v) \to (l'', w)$ 存在。
- **投影**：$p : \mathcal{V} \to \mathcal{L}$ 定义为 $p(l, v) = l$。$p$ 是一个拆分纤维化（cloven fibration）：对 $f : l \to l'$ 和 $(l', u)$，笛卡儿提升为 $(l, f^*(u)) \xrightarrow{\bar{f}} (l', u)$。

此等价由 Grothendieck 定理保证——它是纤维化理论的奠基性结果，将"一族参数化的范畴 $\{V_l\}$ 及其间的函子 $\{f^*\}$"打包为单一的总范畴 $\mathcal{V}$。在偏序纤维化中，伪函子性（$\Phi(g) \circ \Phi(f) \cong \Phi(fg)$）退化为严格等式（因偏序集中态射组合唯一）。

**定义 3.3**（层级指派）。一个函子 $\Sigma : \mathcal{P}(X) \to \mathcal{L}$，满足：对 $S \subseteq T$，存在 $\mathcal{L}$ 中的态射 $\Sigma(S) \to \Sigma(T)$（更小的作用域对应更精细的层级，更大的作用域对应更粗糙、更基础的层级）。

> **注**：作用域大小与层级精细度的单调对应是一种建模约定，适用于"局部约束比全局约束更精细"的场景（如图像中的像素 → 区域 → 拓扑）。在某些应用中这一约定不自然——例如排课系统中"教室不冲突"涉及所有教室（大作用域）却属于最强约束层，而蛋白质折叠中化学键约束是局部的（小作用域）却属于最基础的结构层。对于此类场景，$\Sigma$ 应作为独立的建模选择，不受作用域大小驱动。相关推广讨论见附录 A.11。**当前框架的普适性声称（§1.2 的多领域表格）严格来说仅在"作用域大小与层级精细度单调对应"的场景下完全成立；对不满足此对应的领域，需要 A.11 中讨论的推广方案（尤其是方案 3：将层级指派作为独立建模输入）。**

**定义 3.4**（约束族）。一族映射 $\varphi_S : A(S) \to V_{\Sigma(S)}$，构成一个 **lax 自然变换**：对 $S \subseteq T$，

$$f^*(\varphi_T(a)) \leq_{\Sigma(S)} \varphi_S(\rho_{T,S}(a)) \qquad \forall\, a \in A(T)$$

其中 $f = \Sigma(S \subseteq T) : \Sigma(S) \to \Sigma(T)$，$f^* : V_{\Sigma(T)} \to V_{\Sigma(S)}$。

Lax 自然性（而非严格自然性）的选取是经过审慎考虑的：严格自然性要求局部评分完全由全局评分的拉回决定，这过强；lax 自然性允许每层有独立的约束贡献，同时保证了全局崩溃向局部传播的一致性。

> **注**：$\leq_{\Sigma(S)}$ 的语义含义因纤维而异。在硬 CSP 中为逻辑蕴含；在 WCSP 中对应代价的反序（代价更高 = 序中更低）；在概率 CSP 中为概率的自然序。Lax 条件的统一解读是：粗糙层约束拉回到精细层后的要求，不超过精细层本身施加的约束——精细层可以独立地施加额外的约束。

> **实际可验证性**。Lax 自然性条件要求 $f^*(\varphi_T(a)) \leq \varphi_S(\rho_{T,S}(a))$ 对**所有** $S \subseteq T$ 和**所有** $a \in A(T)$ 成立。对有限域 $D_x$，此条件可通过遍历 $A(T) = \prod_{x \in T} D_x$ 枚举检查，但计算量与 $|A(T)| = \prod_{x \in T} |D_x|$ 成正比——对大作用域是指数级的。对连续域（如 §6 中 $D_{(i,j)} = [0,255]^3$），枚举不可行，需要**解析证明**：通常利用 $\varphi_S$ 和 $f^*$ 的具体函数形式推导不等式。§6 的图像实例中，像素层约束 $\varphi_{\{p_i\}}(v) = v/255$ 和区域层约束 $\varphi_R(v_1,v_2) = |v_1 - v_2|$ 的 lax 自然性应通过显式不等式推导验证（而非仅在特定赋值上数值检查），这是实例化时不应省略的步骤。在建模 DSL 中，建议对有限域自动枚举验证，对连续域要求用户提供解析证明或置信度标记。

> **满足 lax 自然性的两条建模路线**。实践中，"随手写"的分层约束往往不自动满足 lax 自然性。为降低建模负担，推荐以下两条系统化的构造路线：
>
> **路线 A：派生式构造（推荐）**。先为粗糙层定义约束 $\varphi_T$，再**由 $\varphi_T$ 派生**精细层约束：$\varphi_S(a_S) := f^*(\varphi_T(a_S^{\mathrm{ext}})) \otimes_{\Sigma(S)} \varphi_S^{\mathrm{local}}(a_S)$，其中 $a_S^{\mathrm{ext}}$ 是 $a_S$ 的某种标准扩展（如零填充、默认值填充），$\varphi_S^{\mathrm{local}}$ 是精细层独立附加的细化项（满足 $\varphi_S^{\mathrm{local}} \geq \top_{\Sigma(S)}$）。此定义下，lax 自然性由构造**自动保证**——精细层约束本身就是"粗糙层拉回 + 额外细化"的形式，无需事后验证。此路线将 lax 自然性从"需要验证的条件"转化为"构造性地保证的性质"，是推荐的默认建模方式。
>
> **路线 B：自由式构造 + 置信度标记**。允许约束族不满足严格的 lax 自然性，但引入置信度标记以区分不同情形。为每个约束 $(S, \varphi_S)$ 附加标记 $\gamma_S \in \{\text{派生}, \text{独立-已验证}, \text{独立-未验证}\}$。对"独立-未验证"约束，截面化闭包 $\mathrm{cl}$ 的修正应被解释为**建模不一致性的诊断信号**（而非纯粹的结构性信息），求解器可据此向建模者报告潜在问题。此路线牺牲了理论保证的整洁性，但在快速原型和探索性建模中更灵活。
>
> **在工程实现（DSL/求解器）中，路线 A 应作为强制默认**：所有精细层约束必须由粗糙层约束经 $f^*$ 拉回加局部细化项的形式构造，lax 自然性由构造自动保证，无需用户手动验证。路线 B 仅作为研究/快速原型模式提供，且求解器应对路线 B 产生的约束自动标记警告。对连续域或约束来源异构的场景，路线 B 可作为**临时**备选，但建模者应以将其迁移到路线 A 为目标。

**定义 3.5**（纤维化 CSP 实例）。一个四元组：

$$\mathfrak{C} = \big(A : \mathcal{P}(X)^{op} \to \mathbf{Set},\quad p : \mathcal{V} \to \mathcal{L},\quad \Sigma : \mathcal{P}(X) \to \mathcal{L},\quad \varphi\big)$$

### 3.3 评分与求解

给定全局赋值 $a \in A(X)$：

**(S1)** 逐层直接评分：$\mathrm{Dir}_l(a) = \bigotimes_l \{\varphi_S(\rho_{X,S}(a)) : \Sigma(S) = l\}$

**(S2)** 涌现贡献：$\mathrm{Em}_l(a) = \bigotimes_l \{f_!(\mathrm{Dir}_{l'}(a)) : f : l' \to l\}$

即：层 $l$ 的涌现贡献来自所有比 $l$ 更精细的层 $l'$ 的直接评分经推前函子 $f_!$ 推到 $l$ 后的综合。类型检查：$\mathrm{Dir}_{l'}(a) \in V_{l'}$，$f_! : V_{l'} \to V_l$，故 $f_!(\mathrm{Dir}_{l'}(a)) \in V_l$，$\bigotimes_l$ 在 $V_l$ 中运算。（由于 $\mathcal{L}$ 是偏序集/薄范畴，从 $l'$ 到 $l$ 至多一条态射，故 $f_!$ 的选取无歧义。）

> **关于 (S2) 中态射集合的建模选择**。(S2) 对层 $l$ 汇集**所有**态射 $f:l'\to l$（即所有精细于 $l$ 的层 $l'$）的推前贡献。在深度大于 2 的偏序 $\mathcal{L}$ 中，链 $l'' \to l' \to l$ 存在时 $l'' \to l$ 也同时存在。此处**不存在重复计数**：$\mathrm{Dir}_{l''}(a)$ 仅包含层级指派 $\Sigma(S) = l''$ 的约束评分，$\mathrm{Dir}_{l'}(a)$ 仅包含 $\Sigma(S) = l'$ 的约束评分——二者推前的是**不同约束集合**在不同层级上的评分，而非同一贡献沿不同路径的多次传递。
>
> 然而，使用全部态射 vs 仅使用 **Hasse 覆盖边**（$l'$ 覆盖 $l$，即二者之间不存在中间层级）是两种不同的**建模语义**：(i) **全部态射**（本文默认）：层 $l$ 直接感受到所有精细层的涌现"远程效应"，适用于涌现效应可跨级传递的场景。(ii) **Hasse 覆盖边**：层 $l$ 仅从直接相邻层接收涌现，更远层的效应需逐级传递（经 $f_!$ 的复合），适用于涌现效应逐级衰减的场景。两种选择在数学上均自洽；选择哪种应根据具体应用的物理语义决定。对 $\mathcal{L}$ 为链的情形（如 §6 的图像实例），两种选择等价（仅有覆盖边）。在实例化非链 $\mathcal{L}$ 时，建模者应明确此选择并论证其合理性。

**(S3)** 总评分：$\mathrm{Score}_l(a) = \mathrm{Dir}_l(a) \otimes_l \mathrm{Em}_l(a)$

**(S4)** **截面化投影算子**（下文简称"闭包"）。给定原始评分剖面 $\tilde{s}(l) = \mathrm{Score}_l(a)$，定义截面化算子 $\mathrm{cl}$，将任意剖面投影为满足截面相容性的 upper-lax 截面（$s(l) \leq_l f^*(s(l'))$，对所有 $f : l \to l'$）。对一般有限偏序 $\mathcal{L}$，按逆拓扑序（从最粗糙层到最精细层）递推：

> **术语说明**：$\mathrm{cl}$ 是**收缩算子**（$\mathrm{cl}(\tilde{s}) \leq \tilde{s}$），而非经典序论中的闭包算子（闭包通常指扩张的 $\mathrm{cl}(\tilde{s}) \geq \tilde{s}$）。严格的序论术语应为"interior"或"kernel"。本文沿用"闭包"一词有两个理由：(1) 它是将剖面空间投影到截面子空间上的**闭包运算**（截面子空间在闭包意义下是闭集）；(2) 它是幂等收缩映射——在投影意义上"施加截面约束后不再变化"，即截面条件在此算子下"封闭"。读者应注意 $\mathrm{cl}$ 的单调收缩性而非扩张性。

$$\mathrm{cl}(\tilde{s})(l) = \tilde{s}(l) \wedge_l \bigwedge_{f : l \to l'} f^*(\mathrm{cl}(\tilde{s})(l'))$$

其中最粗糙层（$\mathcal{L}$ 中的极大元）$l_{\max}$：$\mathrm{cl}(\tilde{s})(l_{\max}) = \tilde{s}(l_{\max})$。当 $\mathcal{L}$ 为链 $l_1 \to \cdots \to l_k$（精细到粗糙）时，递推简化为：$s(l_k) = \tilde{s}(l_k)$，$s(l_i) = \tilde{s}(l_i) \wedge_{l_i} f_i^*(s(l_{i+1}))$。

> **计算注：一般偏序上的递推**。当 $\mathcal{L}$ 为链时，$\mathrm{cl}$ 的递推方向明确（从粗到细），一趟即可完成。当 $\mathcal{L}$ 是一般有限偏序（含菱形、金字塔等结构）时，递推按任意**逆拓扑序**（所有粗糙层先于精细层被处理）进行。由于 $\mathrm{cl}$ 的定义中使用 $\bigwedge_{f:l \to l'} f^*(\mathrm{cl}(\tilde{s})(l'))$——即取所有直接粗糙邻居闭包值的拉回之交——不同的逆拓扑序给出**相同结果**（因为每个 $l$ 被计算时，所有粗于 $l$ 的层已经确定）。因此 $\mathrm{cl}$ 是良定义的，对有限偏序 $\mathcal{L}$ 恰好需要 $|\mathcal{L}|$ 步（每层一步），无需迭代至不动点。这与信念传播在有环图上可能不收敛的情形不同——$\mathrm{cl}$ 的单趟递推是精确的，不是近似。但计算量与 $\mathcal{L}$ 中不可比元素的数量正相关：若 $\mathcal{L}$ 宽度为 $w$（最大反链大小），则每层需汇合至多 $w$ 个拉回值。

截面化闭包 $\mathrm{cl}$ 具有以下性质：

  - **总有定义**：$\mathrm{cl}(\tilde{s})$ 对任何剖面 $\tilde{s}$ 均产生合法的 upper-lax 截面，彻底避免"筛空"风险
  - **收缩性**：$\mathrm{cl}(\tilde{s})(l) \leq_l \tilde{s}(l)$——闭包不增加任何层的评分
  - **幂等性**：$\mathrm{cl}(\mathrm{cl}(\tilde{s})) = \mathrm{cl}(\tilde{s})$
  - **坍缩传播**：若 $\tilde{s}(l') = \bot_{l'}$，则 $f^*(\bot_{l'}) = \bot_l$，故 $\mathrm{cl}(\tilde{s})(l) \leq \bot_l = \bot_l$——坍缩自然沿 $f^*$ 从粗糙层传播到精细层，是闭包不动点性质的直接推论
  - **单调性**：$\tilde{s} \leq \tilde{s}' \implies \mathrm{cl}(\tilde{s}) \leq \mathrm{cl}(\tilde{s}')$

赋值 $a$ 的截面化评分定义为 $s(a) = \mathrm{cl}(\mathrm{Score}(a))$。

> **设计决策说明**：截面化闭包使用 **upper-lax** 方向（$s(l) \leq_l f^*(s(l'))$），即精细层评分不超过粗糙层评分的拉回。这与约束族的 lax 自然性（定义 3.4，$f^*(\varphi_T(a)) \leq \varphi_S(\rho(a))$，lower-lax）方向相反，但二者服务于不同目的：约束族的 lower-lax 保证"精细层约束不弱于粗糙层约束的拉回"；截面的 upper-lax 保证"精细层评分不脱离粗糙层的结构性约束"。正是 upper-lax 方向使坍缩传播成立——粗糙层 $= \bot$ 迫使精细层 $\leq f^*(\bot) = \bot$。
>
> **$\mathrm{cl}$ 的语义偏置与适用边界**：$\mathrm{cl}$ 不是"中立的数学整理"，它本身内嵌了一种特定的层间语义——**粗糙层对精细层具有前提性优先权**（硬上界式压制）。这一语义在本文的核心场景（§1.3 所述的"前提性依赖"）中恰好是期望的设计：拓扑层崩溃后区域层约束"无意义"。然而，对于粗糙层仅表达偏好而非前提的场景（如区域连贯性只是"期望的"而非"必须的"），$\mathrm{cl}$ 的硬上界投影可能过于激进，导致"闭包性坍缩"（附录 A.12）——精细层原始评分尚未坍缩，但经闭包后被粗糙层的低评分拉至 $\bot_l$。这不是框架的数学缺陷（底元素截面始终合法），但可能导致评分丧失信息量。对于需要"软前提"语义的场景，可引入**参数化闭包** $\mathrm{cl}_\lambda$：
>
> $$\mathrm{cl}_\lambda(\tilde{s})(l) = \tilde{s}(l) \wedge_l \lambda_l\big(f^*(\mathrm{cl}_\lambda(\tilde{s})(l'))\big)$$
>
> 其中 $\lambda_l : V_l \to V_l$ 为松弛算子（如 $\lambda_l(v) = v \vee_l \epsilon_l$，将粗糙层上界"抬高" $\epsilon_l$），$\lambda_l = \mathrm{id}$ 时退化为标准 $\mathrm{cl}$。此参数化使建模者可在"硬前提"（$\lambda = \mathrm{id}$）与"软偏好"（$\lambda$ 非平凡）之间连续调节粗糙层的约束强度。参数化闭包的幂等性与单调性条件的严格分析留作后续工作。
>
> **与传播算法的联系**：截面化闭包的递推计算可自然解释为**自顶向下的约束传播**：从最粗糙（最基础）层出发，逐层向精细层传播结构性约束。经典约束传播算法（弧一致性、信念传播）在此框架下对应于对闭包的单调迭代逼近其不动点。
>
> **从 assignment-level 到 domain-level 的传播**（局部更新规则模板）。上述 $\mathrm{cl}$ 是对**单个赋值 $a$** 的评分剖面操作的——要将其提升为可对变量域进行剪枝的求解器组件，需要以下桥接构造：
>
> 定义变量 $x$ 在层 $l$ 上的**域投影评分**为 $\sigma_l(x, d) = \bigoplus_{a \in A(X) : a(x)=d} s_l(a)$，即层 $l$ 上所有使 $x$ 取值 $d$ 的赋值的截面化评分的聚合（$\bigoplus_l$-最优值）。**层间域剪枝规则**定义为：
>
> $$d \in D_x \text{ 可安全删除} \quad\iff\quad \exists\, l \in \mathcal{L},\ \sigma_l(x, d) = \bot_l$$
>
> 即：若存在某层使得"$x$ 取值 $d$ 的所有赋值在该层截面化评分均坍缩"，则 $d$ 可从 $D_x$ 中删除。这等价于：$d$ 不出现在任何非坍缩截面中。
>
> 此规则的**局部近似**（避免枚举 $A(X)$）为纤维化弧一致性（Fibered Arc Consistency, FAC）：
>
> **FAC 规则**：对每个约束 $(S, \varphi_S)$（$\Sigma(S) = l$）、每个变量 $x \in S$、每个 $d \in D_x$，检查是否存在**支撑赋值** $a_S \in A(S)$，满足 $a_S(x) = d$ 且 $\mathrm{cl}(\mathrm{Score}(a_S^*))(l) \neq \bot_l$（其中 $a_S^*$ 是 $a_S$ 的任意全局扩展）。若对**所有**同层约束均无支撑，则删除 $d$。当 $\mathcal{L} = \mathbf{1}$（单层硬 CSP）时，此规则退化为经典 AC-3：对每个约束 $\varphi_S$、每个 $x \in S$、每个 $d \in D_x$，检查是否存在 $a_S$ 使 $a_S(x) = d$ 且 $\varphi_S(a_S) = \top$。
>
> FAC 的传播循环：(1) 自顶向下计算 $\mathrm{cl}$ 传播坍缩；(2) 用剪枝规则删除域值；(3) 域缩减后更新受影响的约束评分；(4) 重复直到不动点。此循环的单调性由 $\mathrm{cl}$ 的单调性和域缩减的单调性保证。精细层的域剪枝通过 $f_!$ 传播到粗糙层的涌现贡献，从而可能触发粗糙层的进一步剪枝——这是经典 AC 中不存在的**跨层传播效应**。
>
> **局部化与增量化**。上述 FAC 定义中"$a_S^*$ 是 $a_S$ 的任意全局扩展"的存在性量词在语义上等价于 AC-3 中"检查是否存在支撑赋值"——退化到单层硬 CSP 时代价一致。但在大规模实例中，逐次全层重算 $\mathrm{cl}$ 是不必要的。对链结构 $\mathcal{L}$，$\mathrm{cl}$ 的递推是单趟的（§3.3 计算注），因此每次约束更新只需重新计算其**下游层的局部区间**（从被更新约束所在层向精细层方向传播），而非全层全算。具体地：当某约束 $(S, \varphi_S)$（$\Sigma(S) = l$）的评分因域缩减而变化时，仅需更新 $\mathrm{Score}_l$ 中受影响的项、重算 $\mathrm{cl}$ 在 $l$ 及其所有精细于 $l$ 的层上的值。这给出了 FAC 的**增量版本**，将传播代价从 $O(|\mathcal{L}|)$（全层）降至 $O(\text{受影响子链长度})$。此外，路线 A 中的标准扩展 $a_S^{\mathrm{ext}}$（零填充/默认值填充）可将"任意全局扩展"的存在性检查替换为**确定性的单次评估**，从而将 FAC 从"带全局阴影的判定规则"转化为更接近消息传递语义的**局部传播算法**。增量 FAC 的完整形式化与复杂度分析留作后续工作。

**(S5)** 解空间为 $A(X)$ 上由截面化评分 $s(a) = \mathrm{cl}(\mathrm{Score}(a))$ 的逐点偏序诱导的 **Pareto 前沿**。

> **关于解的选择原则**：Pareto 前沿是数学上最一般的解概念，但在层数较多时可能指数增长。实际应用中可根据需求选择以下特化方案：(i) **字典序**（$\mathcal{L}$ 为全序时最自然）：按层级优先排序，粗糙层（基础约束）优先于精细层，最实用；(ii) **加权标量化**：为每层指定权重 $w_l$，定义全局评分 $\mathrm{Agg}(a) = \sum_l w_l \cdot \phi_l(s_l(a))$（$\phi_l$ 为纤维到 $\mathbb{R}$ 的嵌入）；(iii) **$\epsilon$-Pareto**：保留近似前沿，控制输出规模。当 $\mathcal{L}$ 为全序且采用字典序时，Pareto 前沿退化为唯一最优解，恢复经典行为。Pareto 前沿作为理论上的完整解概念保留，具体选择原则视应用而定。

## 4 核心机制分析

### 4.1 坍缩：截面不存在性的几何推论

**命题 4.1**（向下坍缩）。设 $f : l \to l'$（$l$ 更精细，$l'$ 更粗糙），$f^*$ 保持底元素。若截面 $s \in \Gamma(\mathcal{L}, \mathcal{V})$ 满足 $s(l') = \bot_{l'}$（粗糙层坍缩），则 $s(l) = \bot_l$（精细层也坍缩）。

*证明*。截面相容性要求 $s(l) = f^*(s(l')) = f^*(\bot_{l'}) = \bot_l$，最后一步由 $f^*$ 保持底元素。

坍缩传播不是外部规则，而是截面存在的必要条件。这完全类似于向量丛中零截面的延拓性质：纤维的局部退化通过丛的整体结构传播到相邻纤维。

**命题 4.1'**（向上坍缩，可选）。若进一步要求 $f^*$ 反映底元素（条件 V2'），则方向相反的坍缩也成立：若 $s(l) = \bot_l$（精细层坍缩），则 $s(l') = \bot_{l'}$（粗糙层也坍缩）。

*证明*。$f^*(s(l')) = s(l) = \bot_l$，由反映底元素得 $s(l') = \bot_{l'}$。

向上坍缩描述了"精细层全面崩溃逼迫粗糙层也崩溃"的情形，这在部分应用中合理（如所有像素都无效则区域结构无意义），但并非普遍需要。

**推论 4.2**（坍缩传递性）。若 $l \to l' \to l''$（从精细到粗糙）且 $s(l'') = \bot_{l''}$，则 $s(l) = \bot_l$。这由函子组合律 $(g \circ f)^* = f^* \circ g^*$ 与 $f^*$、$g^*$ 均保持底元素自动保证。

> **与截面化闭包的联系**：在 §3.3 的闭包框架下，坍缩传播是闭包算子的直接推论。当原始剖面 $\tilde{s}(l') = \bot_{l'}$ 时，$\mathrm{cl}(\tilde{s})(l) = \tilde{s}(l) \wedge_l f^*(\bot_{l'}) = \tilde{s}(l) \wedge_l \bot_l = \bot_l$。坍缩不是算法步骤，而是由闭包的不动点结构自然实现的传播——对应于命题 4.1 在操作层面的实现。

### 4.2 涌现：$f_!$ 的幺半偏差

**定义 4.3**（幺半偏差）。对 $f : l \to l'$，比较映射：

$$\delta_{v,w} : f_!(v \otimes_l w) \longrightarrow f_!(v) \otimes_{l'} f_!(w)$$

（在偏序设定下即 $f_!(v \otimes_l w) \leq_{l'} f_!(v) \otimes_{l'} f_!(w)$）。此映射的方向并非任意约定，而是由伴随与 $f^*$ 的幺半性**典范确定**的（命题 4.4）。当 $\delta$ 为等式时，$f_!$ 幺半，无涌现。当 $\delta$ 为严格不等式时，$f_!$ 非幺半，存在涌现。

**命题 4.4**（$\delta$ 方向的典范性）。设 $f^*$ 为 oplax 幺半函子（$f^*(v \otimes_{l'} w) \geq_l f^*(v) \otimes_l f^*(w)$，含强幺半等式为特例）且 $f_! \dashv f^*$。则 $f_!$ 典范地为 **oplax 幺半函子**：

$$f_!(v \otimes_l w) \leq_{l'} f_!(v) \otimes_{l'} f_!(w), \qquad \forall\, v, w \in V_l$$

*证明*。伴随的 unit 给出 $v \leq_l f^* f_!(v)$，$w \leq_l f^* f_!(w)$。由 $\otimes_l$ 保序：

$$v \otimes_l w \leq_l f^* f_!(v) \otimes_l f^* f_!(w) \leq_l f^*(f_!(v) \otimes_{l'} f_!(w))$$

第二个不等式由 $f^*$ 的 oplax 幺半性（取 $u_i = f_!(v_i)$ 得 $f^*(u_1 \otimes_{l'} u_2) \geq f^*(u_1) \otimes_l f^*(u_2)$，反向使用）。由伴随翻转得 $f_!(v \otimes_l w) \leq_{l'} f_!(v) \otimes_{l'} f_!(w)$。

> **注**：旧版本要求 $f^*$ 为强幺半函子（等式），但证明中仅使用了 $f^*(u_1 \otimes u_2) \geq f^*(u_1) \otimes f^*(u_2)$（oplax 方向）。因此命题自然推广到 oplax $f^*$。当 $f^*$ 仅为 **lax** 幺半（$f^*(u_1 \otimes u_2) \leq f^*(u_1) \otimes f^*(u_2)$，不等号反向）时，证明失效——$f_!$ 的偏差方向不再由伴随结构典范确定（详见附录 A.2）。

**直觉**：为理解 oplax 不等式的涌现含义，需区分两侧的语义。左侧 $f_!(v \otimes_l w)$ 是将精细层约束的**联合效应**整体推到粗糙层；右侧 $f_!(v) \otimes_{l'} f_!(w)$ 是先将各约束**独立推前**再在粗糙层组合。不等式 $f_!(v \otimes_l w) \leq_{l'} f_!(v) \otimes_{l'} f_!(w)$ 表明联合推前在序意义上**不超过**独立推前后组合的结果。关键在于 $\leq$ 的语义因纤维而异：在代价语义（$[0,\infty]^{op}$，$\leq$ 为反自然序）下，这意味着联合推前的代价**更高**——精细层约束之间的相互作用在独立推前时被遗漏，产生了额外的集体代价；在概率语义（$[0,1]$，$\leq$ 为自然序）下，这意味着联合推前的概率**更低**——精细层约束的相互作用使可行性更为受限。两种解读统一为：**当 $\delta$ 严格时，独立推前无法捕获精细层约束之间的交互效应——这一不可分解的交互效应正是涌现的数学签名。** $f_!$ 幺半（$\delta$ 为等式）意味着整体可精确分解为部分之和，无集体效应。

> **关于"涌现"术语的精确范围**。本文将 $f_!$ 的非幺半性称为"涌现"，但应明确其含义边界。日常用法中"涌现"通常指"宏观层面出现微观层面没有的质性新现象"，这是一个本体论断言。本文的"涌现"是一个**操作性定义**：它刻画的是"精细层约束的联合效应在粗粒化（$f_!$）后不可分解为各约束独立贡献之和"——即**不可分解的层间交互效应**。这是涌现概念在约束组合学中的形式化投影，而非涌现的完整哲学定义。特别地，$f_!$ 非幺半是涌现存在的**必要条件**（它检测到了不可分解性），但不充分——真实系统中的涌现还可能涉及动力学、时间尺度分离等本文未建模的要素。

> **注**：反方向的不等式 $f_!(v) \otimes_{l'} f_!(w) \leq_{l'} f_!(v \otimes_l w)$（lax 幺半）一般**不**成立。若其额外成立，则 $f_!$ 为强幺半函子。此反方向在何种结构条件下成立的讨论见附录 A.9。

> **范畴论背景I：教条伴随定理**。命题 4.4 并非即兴引理，而是 Kelly (1974) **教条伴随定理**（doctrinal adjunction theorem）在完备幺半格设定下的特化。一般定理断言：设 $\mathcal{M}, \mathcal{N}$ 为幺半范畴，$R : \mathcal{N} \to \mathcal{M}$ 为 oplax 幺半函子且 $L \dashv R$，则 $R$ 的 oplax 结构典范地诱导 $L$ 上的 oplax 幺半结构（更一般地，定理适用于任何 2-monad $T$ 上的 $T$-代数之伴随）。在我们的设定中，$R = f^*$，$L = f_!$，幺半范畴为完备幺半格。偏序设定下相干性（coherence）约束自动平凡（所有图表至多一条态射，故自动交换），因此命题 4.4 的直接证明与教条伴随定理的一般证明结构相同但更简洁。将此结果识别为教条伴随的实例有两个意义：(1) 它保证了推广到非偏序纤维（$\mathcal{V}_l$ 为一般幺半范畴，含非平凡态射和相干性约束）时，$f_!$ 的 oplax 幺半结构**仍然由理论保证**——不需要重新证明；(2) 它将涌现($\delta \neq 0$)精确定位为教条伴随在幺半结构上的**非退化性**——教条伴随总给出 oplax 结构，涌现等价于此 oplax 结构不可提升为强幺半。

在每根纤维具有度量结构的特殊情形下，涌现强度可逐纤维量化：

$$\|\delta_f\|_{l'} = \sup_{v, w \in V_l} d_{l'}\big(f_!(v \otimes_l w),\; f_!(v) \otimes_{l'} f_!(w)\big)$$

其中 $d_{l'}$ 是 $V_{l'}$ 上与序相容的度量。注意不同纤维上的 $\|\delta_f\|$ 因度量空间不同而**不可直接跨纤维比较**（见附录 A.10）。

### 4.3 解空间的 Pareto 结构

截面范畴 $\Gamma(\mathcal{L}, \mathcal{V})$ 上的序是逐点偏序：

$$s \leq s' \iff \forall\, l \in L,\ s(l) \leq_l s'(l)$$

这是偏序而非全序，因此两个赋值可能不可比。解空间是一个 **Pareto 前沿**。

这反映了多层级约束问题的本质：拓扑层的失败与区域层的失败是不同种类的失败，不应被强行排序。当所有纤维为全序且坍缩严格传播时，Pareto 前沿退化为字典序最优，恢复经典的层级优先求解。

> **$\mathrm{cl}$ 对 Pareto 结构的影响**。截面化闭包 $\mathrm{cl}$ 是收缩算子，它可能改变赋值之间的 Pareto 支配关系：设闭包前 $a$ 不被 $a'$ 支配（$\exists\, l,\ \tilde{s}_l(a) >_l \tilde{s}_l(a')$），闭包后可能 $s_l(a) \leq_l s_l(a')$ 对所有 $l$ 成立（$a$ 变为被 $a'$ 支配）。这是因为 $\mathrm{cl}$ 的修正量 $\Delta_l(a)$ 依赖于赋值 $a$ 在粗糙层的表现——层间耦合使得精细层的"优势"可能在闭包后被粗糙层的"劣势"所抵消。等价地说：**闭包后的 Pareto 前沿是闭包前 Pareto 前沿的子集**（$\mathrm{cl}$ 的收缩性 + 单调性保证：若 $a$ 在闭包后被支配，则 $a$ 不在闭包后的前沿中）。$\mathrm{cl}$ **不会**制造新的不可比性——它只会消除不可比性（通过将某些赋值拉到被支配位置）。因此闭包使 Pareto 前沿**不增大**，这对工程实用性是有利的。

### 4.4 对比：三种机制的统一

| 机制 | 手动补丁版 | 纤维化版 |
|------|------------------|---------|
| 坍缩 | 外部算子 $\kappa$ + 手写公理 | 截面化闭包 + $f^*$ 保持 $\bot$（+ 可选 V2' 反映 $\bot$） |
| 涌现 | ad hoc 算子 $\varepsilon$ + 手写公理 | $f_!$ 的非幺半性（偏差 $\delta$） |
| 传递性 | 公理 1 | 函子组合律 $(gf)^* = f^*g^*$ |
| 一致性 | 公理 2 | 伴随的自然性 $f_! \dashv f^*$ |
| 恒等性 | 公理 3 | $\mathrm{id}^* = \mathrm{id}$ |

**绝大部分手写公理被纤维化的内在性质所吸收。** 保留的结构性条件（$f^*$ 保持 $\bot$ 和 $\bigwedge$、$f^*$ 的 oplax 幺半性）是对纤维化的温和约束，而非独立的一致性公理；它们在许多自然的纤维化实例中自动成立。

## 5 退化与统一

**命题 5.1**。纤维化 CSP 框架在以下特化条件下退化为已知理论：

| 条件 | 退化结果 |
|------|---------|
| $\mathcal{L} = \mathbf{1}$，$\mathcal{V}_* = \mathbf{2}$ | 经典硬 CSP |
| $\mathcal{L} = \mathbf{1}$，$\mathcal{V}_* = ([0,\infty]^{op}, +, \inf)$ | WCSP |
| $\mathcal{L} = \mathbf{1}$，$\mathcal{V}_* = ([0,1], \times, \sup)$ | 概率 CSP |
| $\lvert\mathcal{L}\rvert > 1$，所有 $f^*, f_!$ 平凡 | 多准则 CSP（Pareto 优化） |
| $\lvert\mathcal{L}\rvert > 1$，$f^*$ 非平凡，$f_!$ 幺半，$\mathcal{L}$ 全序 | 字典序优化 CSP $^\dagger$ |
| $\lvert\mathcal{L}\rvert > 1$，$f^*$ 非平凡，$f_!$ 非幺半 | **分层涌现 CSP（新）** |

> $^\dagger$ 字典序退化要求 $\mathcal{L}$ 为**全序**（线性序），使 Pareto 偏序退化为字典全序。当 $\mathcal{L}$ 仅为偏序时，即使 $f_!$ 幺半，解空间仍为 Pareto 前沿而非单一最优解。

## 6 实例：图像的纤维化 CSP 模型

将引言中的图像例子完整实例化：

$$\mathcal{L} = \{\text{像素} \xrightarrow{f_1} \text{区域} \xrightarrow{f_2} \text{拓扑}\}$$

态射方向从精细到粗糙：像素（最精细）→ 区域 → 拓扑（最粗糙/最基础）。

$$\mathcal{V}_{\text{拓扑}} = (\mathbf{2}, \wedge, \vee), \qquad \mathcal{V}_{\text{区域}} = ([0,\infty]^{op}, +, \inf), \qquad \mathcal{V}_{\text{像素}} = ([0,1], \times, \sup)$$

变量 $X = \{(i,j) : \text{像素坐标}\}$，域 $D_{(i,j)} = [0,255]^3$。

约束：

- **拓扑层**：$\varphi_{\text{拓扑}} : A(X) \to \mathbf{2}$，判定像素排列是否符合行列拓扑。全局硬约束。
- **区域层**：$\varphi_{\text{区域},R} : A(R) \to [0,\infty]$，对每个区域 $R$ 度量空间连贯性代价。局部软约束。
- **像素层**：$\varphi_{\text{像素},(i,j)} : A(\{(i,j)\}) \to [0,1]$，单像素先验偏好。极弱约束。

重新索引 $f_2^* : V_{\text{拓扑}} \to V_{\text{区域}}$：将布尔可行性映到代价（$\top \mapsto 0$，$\bot \mapsto \infty$），保持底元素和 $\bigwedge$，强幺半。$f_1^* : V_{\text{区域}} \to V_{\text{像素}}$：将区域代价归一化为像素偏好值（可选强幺半如 §6.1，或 oplax 幺半如 §6.2），保持底元素和 $\bigwedge$。推前 $(f_1)_! : V_{\text{像素}} \to V_{\text{区域}}$ 由 $f_1^*$ 的左伴随自动确定：当 $f_1^*$ 为 oplax 幺半时，$(f_1)_!$ 非幺半——像素间偏好的交互效应在推前时被保留（涌现）。

求解按截面化闭包+Pareto 前沿进行：先确保拓扑层可行（闭包传播坍缩——若拓扑层 $\bot$，则 $f_2^*$ 保持 $\bot$ 使区域层坍缩，进而 $f_1^*$ 使像素层坍缩），在可行赋值中最小化区域代价，最后在 Pareto 前沿上微调像素偏好。

### 6.1 微型计算验证

为展示评分流水线的完整运作，将上述模型特化到 $2 \times 1$ 图像（两个像素 $p_1, p_2$），简化为两层 $\mathcal{L} = \{\text{像素} \xrightarrow{f} \text{区域}\}$。

**纤维与函子**：$V_{\text{像素}} = ([0,1], \times, \sup)$，$V_{\text{区域}} = ([0,\infty]^{op}, +, \inf)$。取 $f^*(c) = e^{-c/255}$（强幺半：$f^*(c_1 + c_2) = e^{-(c_1+c_2)/255} = f^*(c_1) \times f^*(c_2)$；保持底：$f^*(\infty) = 0 = \bot_{\text{像素}}$；保持 $\bigwedge$：连续递减函数将 $\sup$ 映为 $\inf$），其左伴随 $f_!(p) = -255\ln p$（$p > 0$），$f_!(0) = \infty$。

> 验证伴随：$f_!(p) \leq_{\text{区域}} c$ $\iff$ $-255\ln p \geq c$ $\iff$ $p \leq e^{-c/255} = f^*(c)$。$\checkmark$
>
> 此例中 $f_!$ 恰好幺半：$f_!(p_1 \times p_2) = -255\ln(p_1 p_2) = f_!(p_1) + f_!(p_2)$。这是因为 $f^*$ 是强幺半同构（$\exp$ 与 $\ln$ 的对偶性）。当 $f^*$ 从强幺半放宽到非平凡的 oplax 幺半时，$f_!$ 将不再幺半——§6.2 给出了一个完整的非幺半实例，其中 $f_!$ 严格由伴随结构导出。

**约束**：像素层 $\varphi_{\{p_i\}}(v) = v/255$（亮度偏好），区域层 $\varphi_{\{p_1,p_2\}}(v_1,v_2) = |v_1 - v_2|$（连贯性代价）。域 $D = \{50, 128, 200, 255\}$。

**赋值 $a = (128, 200)$ 的完整评分**：

| 步骤 | 像素层 | 区域层 |
|------|--------|--------|
| (S1) $\mathrm{Dir}_l$ | $\frac{128}{255} \times \frac{200}{255} \approx 0.394$ | $\lvert 128 - 200 \rvert = 72$ |
| (S2) $\mathrm{Em}_l$ | $\top_{\text{像素}} = 1$（无更精细层） | $f_!(0.394) = -255\ln(0.394) \approx 237.6$ |
| (S3) $\mathrm{Score}_l$ | $0.394 \times 1 = 0.394$ | $72 + 237.6 = 309.6$ |

截面化闭包 (S4)：$\mathrm{cl}(\tilde{s})(\text{区域}) = 309.6$（最粗糙层不变），$\mathrm{cl}(\tilde{s})(\text{像素}) = \min(0.394,\; f^*(309.6)) = \min(0.394,\; e^{-309.6/255}) = \min(0.394,\; 0.296) = 0.296$。闭包将像素层评分从 $0.394$ 修正为 $0.296$——粗糙层的高代价通过 $f^*$ 向精细层传播了结构性约束。

**赋值 $a' = (200, 200)$ 的完整评分**：

| 步骤 | 像素层 | 区域层 |
|------|--------|--------|
| (S1) $\mathrm{Dir}_l$ | $\frac{200}{255} \times \frac{200}{255} \approx 0.615$ | $\lvert 200 - 200 \lvert = 0$ |
| (S2) $\mathrm{Em}_l$ | $1$ | $f_!(0.615) \approx 123.8$ |
| (S3) $\mathrm{Score}_l$ | $0.615$ | $123.8$ |

截面化闭包：$\mathrm{cl}(\tilde{s})(\text{像素}) = \min(0.615,\; f^*(123.8)) = \min(0.615,\; 0.615) = 0.615$。取等，无修正。$\checkmark$

**Pareto 比较**：闭包后，$a'$ 在两层上均优于 $a$（像素层 $0.615 > 0.296$，区域层 $123.8 < 309.6$ 即代价更低），故 $a'$ Pareto 支配 $a$。

**坍缩演示**（三层情形）：若恢复拓扑层并设某赋值使 $\varphi_{\text{拓扑}}(a) = \bot$，则 $\mathrm{Score}_{\text{拓扑}} = \bot$。截面化闭包自顶向下传播：$\mathrm{cl}(\tilde{s})(\text{区域}) = \tilde{s}(\text{区域}) \wedge f_2^*(\bot) = \tilde{s}(\text{区域}) \wedge \infty = \bot_{\text{区域}}$，进而 $\mathrm{cl}(\tilde{s})(\text{像素}) = \tilde{s}(\text{像素}) \wedge f_1^*(\infty) = \tilde{s}(\text{像素}) \wedge 0 = \bot_{\text{像素}}$。所有层同时坍缩——这是截面化闭包的不动点性质，也是命题 4.1 的具体体现。

### 6.2 非幺半 $f_!$ 的微型计算验证

§6.1 中 $f_! = -255\ln(\cdot)$ 恰好幺半，无法展示涌现。本节构造一个 $f_!$ 非幺半的最小实例，完整演算涌现贡献。与旧版本不同，此处的 $f_!$ **严格由 $f^*$ 的 Galois 伴随导出**，确保 $\delta$ 的 oplax 方向由理论保证。

**设定**。仍取两像素 $p_1, p_2$，两层 $\mathcal{L} = \{\text{像素} \xrightarrow{f} \text{区域}\}$，纤维同前：$V_{\text{像素}} = ([0,1], \times, \sup)$，$V_{\text{区域}} = ([0,\infty]^{op}, +, \inf)$。

**改用 oplax 幺半的 $f^*$**。定义 $f^*(c) = e^{-\sqrt{c/255}}$。

验证 (V2) 条件：

- **保持底**：$f^*(\infty) = e^{-\infty} = 0 = \bot_{\text{像素}}$。$\checkmark$
- **保持 $\bigwedge$**：$f^*$ 是 $[0,\infty) \to (0,1]$ 上的连续严格递减函数（视 $[0,\infty]^{op}$ 中 $\bigwedge = \sup_{\mathrm{nat}}$，$[0,1]$ 中 $\bigwedge = \inf_{\mathrm{nat}}$），故 $f^*(\sup_{\mathrm{nat}} S) = \inf_{\mathrm{nat}} f^*(S)$。$\checkmark$
- **oplax 幺半**：$f^*(c_1 + c_2) = e^{-\sqrt{(c_1+c_2)/255}} \geq e^{-\sqrt{c_1/255}-\sqrt{c_2/255}} = f^*(c_1) \times f^*(c_2)$。最后一步由 $\sqrt{a+b} \leq \sqrt{a}+\sqrt{b}$（$a, b \geq 0$）。$\checkmark$
- **非强幺半**（与 §6.1 的关键区别）：取 $c_1 = 50, c_2 = 72$，$f^*(122) = e^{-\sqrt{0.478}} = e^{-0.692} \approx 0.501$，$f^*(50) \cdot f^*(72) = e^{-0.443} \cdot e^{-0.531} \approx 0.642 \times 0.588 = 0.378$。$0.501 \neq 0.378$，严格 oplax。

**由伴随导出 $f_!$**。$f_! \dashv f^*$ 给出 $f_!(p) = \sup\{c : p \leq f^*(c)\} = \sup\{c : p \leq e^{-\sqrt{c/255}}\}$。求解 $p \leq e^{-\sqrt{c/255}}$ 得 $\sqrt{c/255} \leq \ln(1/p)$，即 $c \leq 255(\ln(1/p))^2$。因此：

$$f_!(p) = 255\big(\ln(1/p)\big)^2, \qquad p > 0; \qquad f_!(0) = \infty$$

> 验证伴随：$f_!(p) \leq_{\text{区域}} c$ $\iff$ $255(\ln(1/p))^2 \geq c$ $\iff$ $\ln(1/p) \geq \sqrt{c/255}$ $\iff$ $p \leq e^{-\sqrt{c/255}} = f^*(c)$。$\checkmark$

**验证非幺半性与 $\delta$ 的 oplax 方向**。取 $p_1 = 0.8, p_2 = 0.6$：

- 联合推前：$f_!(0.8 \times 0.6) = f_!(0.48) = 255(\ln(1/0.48))^2 = 255 \times (0.734)^2 \approx 137.4$
- 独立推前后求和：$f_!(0.8) + f_!(0.6) = 255 \times (0.223)^2 + 255 \times (0.511)^2 \approx 12.7 + 66.5 = 79.2$

在 $[0,\infty]^{op}$ 中 $137.4 \leq_{op} 79.2$（即 $137.4 \geq_{\mathrm{nat}} 79.2$），确认 **oplax** 方向 $f_!(p_1 \times p_2) \leq_{op} f_!(p_1) + f_!(p_2)$，且不等式严格——**涌现存在**。

涌现偏差 $\delta = 137.4 - 79.2 = 58.2$，精确等于交叉项 $2 \times 255 \times \ln(1/0.8) \times \ln(1/0.6) = 2 \times 255 \times 0.223 \times 0.511 = 58.1$。这一交叉项的数学来源是 $(\ln(1/p_1) + \ln(1/p_2))^2 = \ln(1/p_1)^2 + 2\ln(1/p_1)\ln(1/p_2) + \ln(1/p_2)^2$ 中的非对角项——**它精确地编码了两个像素偏好之间的交互效应**。在幺半 $f_!$ 中此交叉项为零；在非幺半 $f_!$ 中它捕获了不可分解的集体效应。

> **关键对比**：§6.1 中 $f^* = e^{-c/255}$（强幺半）自动导出 $f_! = -255\ln p$（幺半），$\delta = 0$。本节中 $f^* = e^{-\sqrt{c/255}}$（严格 oplax 幺半）自动导出 $f_! = 255(\ln(1/p))^2$（非幺半），$\delta > 0$。两者的区别**纯粹来自 $f^*$ 的幺半性质差异**——涌现/无涌现的判定由纤维化结构（而非手工选择的聚合器）内在决定。

**完整评分（使用非幺半 $f_!$）**。约束同 §6.1。赋值 $a = (128, 200)$：

| 步骤 | 像素层 | 区域层 |
|------|--------|--------|
| (S1) $\mathrm{Dir}_l$ | $\frac{128}{255} \times \frac{200}{255} \approx 0.394$ | $\lvert 128 - 200 \rvert = 72$ |
| (S2) $\mathrm{Em}_l$ | $\top = 1$ | $f_!(0.394) = 255 \times (0.932)^2 \approx 221.3$ |
| (S3) $\mathrm{Score}_l$ | $0.394$ | $72 + 221.3 = 293.3$ |
| (S4) $\mathrm{cl}$ | $\min(0.394,\; e^{-\sqrt{293.3/255}}) = \min(0.394,\; 0.342) = 0.342$ | $293.3$ |

赋值 $a' = (200, 200)$：

| 步骤 | 像素层 | 区域层 |
|------|--------|--------|
| (S1) $\mathrm{Dir}_l$ | $0.615$ | $0$ |
| (S2) $\mathrm{Em}_l$ | $1$ | $f_!(0.615) = 255 \times (0.486)^2 \approx 60.3$ |
| (S3) $\mathrm{Score}_l$ | $0.615$ | $60.3$ |
| (S4) $\mathrm{cl}$ | $\min(0.615,\; e^{-\sqrt{60.3/255}}) = \min(0.615,\; 0.615) = 0.615$ | $60.3$ |

**涌现效应的对比**。与 §6.1 的幺半 $f_!$ 相比，非幺半 $f_!$ 使区域层涌现贡献从 $237.6$ 变为 $221.3$（对 $a$）、从 $123.8$ 变为 $60.3$（对 $a'$）。直观含义：强幺半 $f^* = e^{-c/255}$ 将代价线性映射为对数偏好（$f_!$ 是 $\ln$ 的线性逆变换），所有交互信息在推前时被完全分解；oplax 幺半 $f^* = e^{-\sqrt{c/255}}$ 将代价以次线性方式映射为偏好（"大代价的影响被压缩"），其推前 $f_! = 255(\ln(1/p))^2$ 的二次结构使像素间的偏好交互在推前时被保留——这正是涌现的数学签名。

## 7 复杂度猜想

### 7.1 陈述

**定理-猜想 7.1**。设 $\mathfrak{C}$ 为纤维化 CSP 实例。以下结果将 $f_!$ 的幺半偏差 $\delta$ 定位为层间复杂度不可分解性的**关键不变量**——它是检测层间交互效应的必要且主导的维度，但不构成整个问题 P/NP 分类的充分判据（层内多态性判据与约束图宽度类条件是正交且互补的维度，见 §8.6）。

**(A-eval)**（评分可分解性——**已证**）。若对所有 $f : l \to l'$，$f_!$ 是幺半函子，则每个赋值 $a \in A(X)$ 的逐层评分函数可独立计算，涌现贡献可逐约束分解。（证明见命题 7.2a。）

**(A-opt)**（优化可分解性——**条件定理**）。若对所有 $f : l \to l'$，$f_!$ 是幺半函子，且满足以下**全部**附加条件：
  - (A-opt-1) $\mathcal{L}$ 为**全序**，求解按**字典序**进行；
  - (A-opt-2) 粗糙层最优赋值集 $A^*_{l_k}$ 在精细层约束下仍有足够自由度——形式化为：对每一步字典序求解，上一层最优集在下一层的投影覆盖下一层近似最优解（**共享赋值空间兼容性条件**，见附录 A.4）；

则优化复杂度可逐层分解：

$$T(\mathfrak{C}) = O\Big(\mathrm{poly}(|\mathcal{L}|) \cdot \sum_{l \in L} T(\mathfrak{C}_l)\Big)$$

> **开放问题**：对一般偏序 $\mathcal{L}$（含不可比元素），即使评分可逐层独立计算，Pareto 前沿大小可能指数增长。是否存在使优化复杂度可分解的额外结构条件（如各层约束作用域具有有界树宽、层间变量交互图具有有界分解宽度）是一个开放问题（见附录 A.4）。当前证据表明，$f_!$ 幺半是优化可分解的**必要但非充分**条件——还需要共享赋值空间上的兼容性或层间交互受限。

**(B)**（弱版本）存在一族实例使得 $f_!$ 非幺半且全局复杂度严格超过逐层之和——具体地，每层多项式可解而全局 NP-hard。

**(B')**（强版本）存在一族实例使得 $f_!$ 非幺半、$f_!$ 的电路复杂度为 $\mathrm{TC}^0$（即 $f_!$ 本身"计算简单"），每层多项式可解，而全局 NP-hard。此版本排除了困难性来自 $f_!$ 自身计算能力的平凡情形，揭示困难性真正来源于非幺半性与约束结构的交互。

> **隐含假设**：定理-猜想 (A) 和 (B) 均假定所有 $f_!$ 和 $f^*$ 是多项式时间可计算的。猜想 (B) 的弱版本允许 $f_!$ 的计算复杂度参与产生困难性，因而可通过直接在 $f_!$ 中编码 NP-hard 实例来平凡地证明（见 §7.3）。强版本 (B') 限制 $f_!$ 的描述复杂度后，需要在伴随合法域内完成非平凡的归约构造（见 §7.3 及附录 A.5）。

> **§7 各结果的成熟度梯度**。本节各结果的严格程度差异显著，读者应据此校准期望：
>
> | 结果 | 状态 | 成熟度 |

### 7.2 可分解方向 (A) 的证明策略

$f_!$ 幺半时的"可分解性"实际涉及两个层次不同的命题，需要严格区分。

**命题 7.2a**（评分可分解性）。设所有 $f_!$ 幺半，则每个赋值 $a \in A(X)$ 的逐层评分函数 $\mathrm{Score}_l(a)$ 可独立计算——涌现贡献 $\mathrm{Em}_l(a)$ 可逐约束分解。

*证明*。在第 $l$ 层，涌现贡献为：

$$\mathrm{Em}_l(a) = \bigotimes_l \{f_!(\mathrm{Dir}_{l'}(a))\}$$

由 $f_!$ 幺半，$f_!(\bigotimes_{l'} v_i) = \bigotimes_l f_!(v_i)$，故涌现贡献可逐约束独立计算推前后再组合。$\square$

**命题 7.2b**（优化可分解性——附加条件下，对应定理-猜想 A-opt 的证明核心）。设所有 $f_!$ 幺半，$\mathcal{L}$ 为**全序**，且求解按**字典序**进行。则优化复杂度可逐层分解：

$$T(\mathfrak{C}) = O\Big(\mathrm{poly}(|\mathcal{L}|) \cdot \sum_{l \in L} T(\mathfrak{C}_l)\Big)$$

*证明思路*。自粗糙层到精细层依次求解：在最粗糙层上求最优赋值集 $A^*_{l_k} \subseteq A(X)$，在下一层上从 $A^*_{l_k}$ 中选子集 $A^*_{l_{k-1}}$ 使第 $l_{k-1}$ 层评分最优，依次进行。每步需要在上一步的最优集内求解一个单层子问题。关键条件是：评分函数的独立计算性（命题 7.2a）保证每步无需重新计算其他层的贡献。**注意**：此证明思路隐含地假设了共享赋值空间兼容性（定理-猜想 A-opt 的条件 A-opt-2）——即粗糙层最优集仍为精细层留出足够自由度。该条件的精确形式化见附录 A.4。$\square$

> **评分可分解 $\neq$ 优化可分解**：命题 7.2a 对一般偏序 $\mathcal{L}$ 均成立；命题 7.2b 需要 $\mathcal{L}$ 全序 + 字典序这两个附加条件。对一般偏序 $\mathcal{L}$，即使评分可逐层独立计算，Pareto 前沿的大小可能指数增长，优化复杂度不保证可分解。更精确地说：
>
> - **评分可分解**：给定赋值 $a$，计算 $\mathrm{Score}_l(a)$ 的总时间 $= O(\sum_l T_{\mathrm{eval}}(\mathfrak{C}_l))$。这是关于**评价**的命题。
> - **优化可分解**：找到最优（或 Pareto 前沿）赋值的总时间可逐层分解。这是关于**搜索**的命题，还需要赋值空间在各层之间具有足够的独立结构。
>
> 当 $\mathcal{L}$ 为偏序且含不可比元素时，是否存在使优化复杂度可分解的额外条件（如各层约束作用域具有有界树宽、或层间 $f^*$ 满足特殊结构），是一个开放问题（见附录 A.4）。

### 7.3 不可分解方向 (B) 的归约构造

**定理 7.3**（不可分解性——弱版本 (B) 的证明）。存在两层纤维化 CSP 实例族，每层多项式可解而全局 NP-hard。

*证明思路*。将 3-SAT 归约为纤维化 CSP：

- **细层**（精细）：$n$ 个布尔变量，仅一元约束。独立求解 $O(n)$。
- **粗层**（粗糙）：无约束。独立求解 $O(n)$。
- **推前 $f_! : V_{\text{细}} \to V_{\text{粗}}$**：编码 3-SAT 子句的交互。一个子句 $(x_i \vee \bar{x}_j \vee x_k)$ 的满足性不能分解为一元函数的 $\otimes$-积。因此 $f_!$ 非幺半。

全局评分中，粗层收到的涌现贡献 $f_!(\mathrm{Dir}_{\text{细}}(a))$ 编码了全部子句的联合满足性。判定 Pareto 可行性等价于 3-SAT 可满足性。

> **注**：此归约将 3-SAT 的困难性完全编码到了 $f_!$ 中——所有复杂性藏在层间推前函子里，使得弱版本 (B) 近乎平凡。这恰恰说明了强版本 (B') 的必要性。

**强版本 (B') 的归约方向**。限制 $f_!$ 的电路复杂度为 $\mathrm{TC}^0$（线性阈值函数的常数深度电路），要求归约在**伴随合法域内**完成——即 $f_!$ 必须是某个满足 (V2) 的 $f^*$ 的左伴随，而非任意手写。修订后的构造（附录 A.5）采用 A.13 的方法一：先选择合法的 oplax 幺半 $f^*$，再由伴随自动导出 $f_!$，然后利用约束作用域结构编码 NAE-3SAT 条件。此路线确保困难性来自非幺半性与约束结构的交互，而非来自不合法的 $f_!$。

### 7.4 精化：涌现宽度

**定义 7.4**（涌现宽度）。对 $f : l \to l'$，定义

$$\mathrm{ew}(f) = \min\Big\{k : \exists\, g_1, \dots, g_k : V_l \to V_{l'}, \text{ 使得 } f_!(v \otimes_l w) = \bigoplus_{j=1}^{k} g_j(v) \otimes_{l'} g_j(w),\ \forall\, v, w \in V_l\Big\}$$

即 $f_!$ 在 $\otimes_l$ 上的行为可以被分解为 $k$ 个"秩一"分量的 $\bigoplus$-聚合。当 $\mathrm{ew}(f) = 1$ 时，取 $g_1 = f_!$ 即恢复幺半性。此定义类比于双线性形式的秩（CP 分解秩），将 $f_!$ 的非幺半性量化为一个有限整数。

> **备注**：此定义的严格化需要解决以下问题。(1) **$g_j$ 的正则性**：若对 $g_j$ 不加限制，则可能通过编码全部交互信息的非标准映射使 $\mathrm{ew}(f) = 1$ 对所有 $f_!$ 平凡成立。为避免此退化，应要求 $g_j$ 为**序保持映射**（保证分解与值域的代数结构相容）。(2) **有限性**：对有限值域 $|V_l| = n$，$\mathrm{ew}(f) \leq n$（至多 $n$ 个秩一分量即可精确表示任意映射），但最优分解数可能远小于 $n$。(3) **可计算性**：计算 $\mathrm{ew}(f)$ 类比于半环上矩阵秩的计算，很可能是 NP-hard 的。(4) **$\bigoplus$ 聚合的收敛性**：对无穷值域需额外假设 $\bigoplus$ 的完备性。详见附录 A.3。

**猜想 7.5**（精化复杂度界——探索性）。

$$T(\mathfrak{C}) = O\Big(\prod_l T(\mathfrak{C}_l) \cdot \prod_{f \in \mathrm{Mor}(\mathcal{L})} |D|^{\mathrm{ew}(f)}\Big)$$

> **注**：此猜想目前是高度探索性的。即使在类比的张量秩理论中，秩与计算复杂度的关系也远非如此简洁。此公式应被视为"涌现宽度参数化复杂度"方向的初步猜测，而非精确断言。

涌现宽度与经典的树宽（treewidth）度量了复杂度的**正交方向**：树宽刻画层内约束的水平交互结构，涌现宽度刻画层间约束的垂直交互结构。经典参数化复杂度理论仅捕获前者。

### 7.5 涌现宽度的可操作近似

精确计算 $\mathrm{ew}(f)$ 很可能是 NP-hard 的（§7.4 备注），但工程应用中往往不需要精确值——一个可多项式时间计算的**上界**即可作为"层间交互强度"的诊断指标。以下给出两种可操作的替代量。

**定义 7.6**（采样偏差指标 $\hat{\delta}$）。对有限值域 $V_l$，定义

$$\hat{\delta}(f) = \frac{1}{|V_l|^2} \sum_{v, w \in V_l} d_{l'}\big(f_!(v \otimes_l w),\; f_!(v) \otimes_{l'} f_!(w)\big)$$

即幺半偏差在所有输入对上的平均值。$\hat{\delta}(f) = 0 \iff f_!$ 幺半（无涌现）。对无穷值域可改用蒙特卡洛采样近似。$\hat{\delta}$ 是 $O(|V_l|^2)$ 时间可计算的（遍历所有输入对），且 $\hat{\delta}(f) > 0 \implies \mathrm{ew}(f) > 1$。它不给出 $\mathrm{ew}(f)$ 的数值上界，但提供了"涌现是否存在以及平均多强"的快速诊断。

**定义 7.7**（谱上界 $\overline{\mathrm{ew}}$）。对有限值域 $|V_l| = n$，将 $\phi(v,w) = f_!(v \otimes_l w)$ 视为 $n \times n$ 矩阵（元素取值于 $V_{l'}$）。在适当嵌入下（如将 $V_{l'}$ 嵌入 $\mathbb{R}$），定义

$$\overline{\mathrm{ew}}(f) = \mathrm{rank}_{\mathbb{R}}(\Phi), \qquad \Phi_{ij} = \phi_{\mathbb{R}}(v_i, v_j) \in \mathbb{R}$$

其中 $\phi_{\mathbb{R}}$ 是 $\phi$ 在实数域上的嵌入（类似于核范数松弛对矩阵秩的替代）。此量满足 $\overline{\mathrm{ew}}(f) \geq \mathrm{ew}(f)$（实数秩不低于半环秩），且可用 SVD 在 $O(n^3)$ 时间内计算。当 $\overline{\mathrm{ew}}(f) = 1$ 时，$f_!$ 在嵌入意义下幺半。

> **使用建议与定位**：$\hat{\delta}$ 和 $\overline{\mathrm{ew}}$ 应被定位为**工程诊断启发式**，而非理论不变量 $\mathrm{ew}(f)$ 的严格替代。其有效性依赖于将纤维嵌入 $\mathbb{R}$ 的选择（不同嵌入可能给出不同的 $\overline{\mathrm{ew}}$ 值），且不同纤维之间的 $\hat{\delta}$ 值因度量空间不同而不可直接跨纤维比较（详见附录 A.10）。在实际建模中，建议先用 $\hat{\delta}(f)$ 快速诊断涌现是否存在及强度量级，再对 $\hat{\delta}$ 较大的态射计算 $\overline{\mathrm{ew}}(f)$ 以获得交互维度的上界。两者均为多项式时间可计算，适合嵌入到建模 DSL 的自动诊断流水线中。精确的 $\mathrm{ew}(f)$ 保留为理论分析工具。特别地，$\hat{\delta}$ 可用于参数化实验：在保持每层子问题难度不变的情况下，调节 $f^*$ 的 oplax 程度（如 A.13 实例族中的参数 $\alpha$），以 $\hat{\delta}$ 量化层间交互强度，观察搜索节点数/剪枝率/运行时间是否随 $\hat{\delta}$ 单调变化——这将直接支撑"层间交互导致难度上升"的核心直觉。

## 8 框架的实质意义

### 8.1 新的结构分类

经典 CSP 的 Bulatov-Zhuk 二分定理以多态性代数为判据，将约束语言分为 P 和 NP-complete。这是单层分析——所有约束在同一个代数中。纤维化框架引入了一个新的维度：同为 NP-hard 的问题，困难性可能来自层内约束交互，也可能来自层间涌现。**经典理论无法区分这两种来源，纤维化框架可以。**

### 8.2 新的不变量

幺半偏差 $\delta$ 和涌现宽度 $\mathrm{ew}$ 是经典框架中没有对应物的量。它们的定义**依赖于纤维化结构的存在**——没有"纤维"和"推前函子"，这些量连陈述都无法给出。

### 8.3 坍缩的几何化

坍缩传播在经典框架中是操作性规则；在纤维化框架中是截面存在性的几何约束。这不仅更优雅，而且揭示了坍缩的本质：它不是算法步骤，而是值域空间的拓扑性质。

### 8.4 跨领域迁移

纤维化 CSP 的核心结构——双纤维化上的截面、$f^*$ 与 $f_!$ 的伴随、$f_!$ 的幺半偏差——不依赖于 CSP 的具体细节。相同的框架适用于具有多层级约束的一般系统。在结构层面值得注意的是与**重整化群（RG）的类比**：$f_!$ 从细层推前到粗层，形式上类似于"积分掉细粒度自由度"；$f_!$ 的非幺半性类似于"积分掉快模式后慢模式之间产生的新有效相互作用"。

> **关于 RG 类比的限制**。此类比目前仅为**结构层面的启发性对应**，而非严格等价，需警惕以下差异：(1) RG 中的粗粒化通常保留配分函数（或自由能），是**精确等价变换**；而 $f_!$ 作为 Galois 连接的左伴随，不保证任何意义上的"等价"——信息在推前过程中不可逆地丢失。(2) RG 具有**自相似/不动点结构**（临界现象），纤维化 CSP 中 $f_!$ 的迭代不具备类似的不动点语义。(3) RG 中粗粒化的对象是概率分布/场构型，而纤维化 CSP 中 $f_!$ 作用于评分值——二者的定义域本质不同。将此类比严格化需要在连续值域和统计力学设定下实例化整个框架（详见附录 A.6），这是后续工作方向而非当前结论。

### 8.5 范畴论在本文中的实际角色

本文标题含"纤维化富集范畴"，有必要坦诚评估范畴论在当前偏序设定下的实际贡献。

**做了实事的**：

1. **Grothendieck 构造**（命题 3.2'）：将分散的数据（一族值域 $\{V_l\}$、一族函子 $\{f^*\}$、一族一致性条件）打包为单一的总范畴 $\mathcal{V}$。打包之后，截面、自然变换等全局概念才有着落。不用 Grothendieck 构造，这些数据必须以公理列表形式逐一维护——旧版本中的手写公理 1-3 正是此情形。**这是纤维化语言相对于公理列表的唯一不可替代贡献。**

2. **教条伴随定理**（§4.2）：命题 4.4（$f^*$ oplax $\implies$ $f_!$ oplax）不是特设引理，而是 Kelly (1974) 的一般定理在偏序上的特化。识别此出处有实际意义：若将来将纤维从偏序推广到含非平凡态射的幺半范畴，教条伴随定理保证 $f_!$ 的 oplax 结构**仍然自动成立**（含完整的相干性条件），无需重新证明。

**仅提供语言便利的**（可用保序映射+不等式完全替代）：

- 函子组合律 $(gf)^* = f^* \circ g^*$——就是映射组合。
- Lax 自然变换（约束族一致性）——就是逐对不等式 $f^*(\varphi_T(a)) \leq \varphi_S(\rho(a))$。
- 截面（全局一致选取）——就是满足一族不等式的元组。

**诚实的结论**：在本文的偏序设定下，范畴论的**实质**贡献集中在两处：Grothendieck 构造提供了"打包"机制，教条伴随定理提供了涌现方向性的理论来源。其余范畴论术语（截面、lax 自然变换、函子组合律等）在偏序设定下等价于保序映射和不等式——选择范畴语言是为了表述的统一性和日后推广的便利性，不是因为偏序设定下不可替代。

### 8.6 与已有工作的关系

本框架涉及多个已有研究方向，有必要明确定位本文的新贡献与已有工作的关系。

**VCSP 与多态性代数理论**。Bulatov (2017) 与 Zhuk (2020) 的 CSP 二分定理以多态性代数（polymorphism algebra）为判据，将约束语言分为 P 和 NP-complete。纤维化框架引入了正交于多态性的新维度：同为 NP-hard 的问题，经典理论无法区分困难性来自层内约束交互还是层间涌现。$f_!$ 的幺半偏差 $\delta$ 和涌现宽度 $\mathrm{ew}$ 是这一新维度上的不变量，其定义依赖于纤维化结构的存在——在经典 VCSP 中连陈述都无法给出。两种分类方式原则上互补而非互斥：多态性刻画层内代数结构，纤维化不变量刻画层间交互结构。**将二者整合为统一的分类系统是重要的开放方向。**

**多目标约束优化（MOCSP）**。当 $\mathcal{L}$ 含不可比元素时，纤维化 CSP 的解空间为 Pareto 前沿（§4.3），这与多目标优化文献（如 Gavanelli, 2002; Lukasiewicz et al., 2014）中的 Pareto CSP 形式上相似。关键区别在于：MOCSP 将多目标视为同一代数中的独立目标函数，目标间无结构性关联；纤维化 CSP 中各"目标"（各层评分）通过 $f^*$ 和 $f_!$ 耦合——坍缩传播和涌现是这种耦合的直接后果，在 MOCSP 中没有对应物。当所有 $f^*, f_!$ 平凡时，纤维化 CSP 退化为 MOCSP（§5 退化表已列出）。

**分层规划与层级求解**。分层约束传播在 AI 规划（hierarchical task networks, Erol et al., 1994）和分层优化（lexicographic optimization）中已有大量工程实践。纤维化框架的贡献不在于"分层求解"的操作思路本身，而在于为层间关系提供了精确的代数语义：(i) "坍缩"不是启发式的优先级规则，而是截面不存在性的几何推论；(ii) "涌现"不是模糊的"整体大于部分"，而是 $f_!$ 非幺半偏差的可计算量。现有分层规划方法可视为纤维化 CSP 的特殊实例（$f_!$ 幺半、$\mathcal{L}$ 全序）。

**格值 CSP**。Rossi et al. (2006) 的 Semiring CSP 和格值 CSP 允许非布尔值域，但仍然是单值域框架——所有约束映射到同一个半环/格。纤维化框架的关键改进是允许**不同约束映射到不同的代数结构**（不同纤维），并通过 $f^*/f_!$ 伴随桥接它们。这一点在单值域框架中没有对应物。

## 附录 A：开放问题与待进一步分析的技术点

本节列出当前框架中已识别但尚未完全解决的技术问题，以指引后续工作。

### A.1 截面化闭包的性质与闭包不修正的充分条件

**背景**：§3.3 引入的截面化闭包 $\mathrm{cl}$ 彻底解决了旧版本 (S4) 的"筛空"风险——闭包对任意评分剖面总能给出合法的 upper-lax 截面。问题转化为：在何种条件下，原始评分剖面**已经是截面**（即闭包不修正任何值）？

**形式化**：$\mathrm{cl}(\tilde{s}) = \tilde{s}$ 当且仅当对所有 $f : l \to l'$，$\tilde{s}(l) \leq_l f^*(\tilde{s}(l'))$（upper-lax 截面条件自动成立）。

**初步分析**：

1. **$f_!$ 幺半时**：当所有 $f_!$ 幺半时，涌现贡献 $\mathrm{Em}_l$ 可逐约束分解。此时由伴随的 unit $v \leq f^* f_!(v)$ 以及 $f^*$ 的 oplax 幺半性，有：$\mathrm{Score}_l(a) = \mathrm{Dir}_l(a) \otimes_l \mathrm{Em}_l(a)$，而 $f^*(\mathrm{Score}_{l'}(a))$ 通过 $f^*$ 的 oplax 性与伴随 unit 的组合，有较好的控制。§6.1 的数值验证确认了此情形下闭包不修正（或仅取等）。猜测：当 $f_!$ 幺半时，$\mathrm{cl}(\mathrm{Score}(a)) = \mathrm{Score}(a)$ 对所有赋值 $a$ 成立。

2. **$f_!$ 非幺半时**：§6.2 的数值验证表明闭包可能修正精细层评分。修正量反映了涌现贡献在层间传播时引入的"不协调性"。修正量的大小与 $f_!$ 的幺半偏差 $\delta$ 正相关。

3. **Beck-Chevalley 条件**：若纤维化满足 Beck-Chevalley 条件（对 $\mathcal{L}$ 中的交换方块，$f^* \circ g_! \cong h_! \circ k^*$），则不同路径计算的推前-拉回组合给出一致的结果，可能使闭包修正量受到更紧的界。此方向需要将 $\mathcal{L}$ 中的交换方块条件实例化到具体的层级结构中。

**闭包修正的语义**：当 $\mathrm{cl}(\tilde{s})(l) < \tilde{s}(l)$（闭包将精细层评分向下修正），其含义是：精细层的原始评分过于"乐观"，未充分反映粗糙层高代价（或低偏好）通过 $f^*$ 传递的结构性约束。闭包修正使评分在层间保持一致——这正是"评分必须尊重纤维化结构"的操作化体现。

### A.2 $f^*$ 幺半性的分类与影响

**问题**：条件 (V2) 要求 $f^*$ 是 oplax 幺半函子（含强幺半为特例）。不同强度的幺半性对框架各机制的影响如何？

在 §6.1 中 $f^*(c) = e^{-c/255}$ 是强幺半同构，$f_!$ 幺半。在 §6.2 中 $f^*(c) = e^{-\sqrt{c/255}}$ 是严格 oplax 幺半，$f_!$ 非幺半。其他自然的归一化方式（如 sigmoid $f^*(c) = 1/(1+c)$）满足 $f^*(c_1 + c_2) = 1/(1+c_1+c_2) \leq 1/(1+c_1) \times 1/(1+c_2)$——这是 **lax** 幺半（不等号方向与 oplax 相反）。

**三种幺半性类型的影响分析**：

| 类型 | 条件 | 命题 4.4($\delta$ 方向) | 坍缩传播 | 适用场景 |
|------|------|----------------------|---------|---------|
| 强幺半 | $f^*(u \otimes w) = f^*(u) \otimes f^*(w)$ | $f_!$ oplax ✓ | ✓ | $f_!$ 可能幺半 |
| Oplax 幺半 | $f^*(u \otimes w) \geq f^*(u) \otimes f^*(w)$ | $f_!$ oplax ✓ | ✓ | §6.2，涌现场景 |
| Lax 幺半 | $f^*(u \otimes w) \leq f^*(u) \otimes f^*(w)$ | **证明失效** ✗ | ✓ | 需额外分析 |

**Lax 幺半 $f^*$ 的问题**：命题 4.4 的证明关键步骤为

$$f^* f_!(v) \otimes f^* f_!(w) \leq f^*(f_!(v) \otimes f_!(w))$$

这需要 $f^*$ 的 **oplax** 方向。若 $f^*$ 是 lax 幺半，此不等式反向（$f^*(u \otimes w) \leq f^*(u) \otimes f^*(w)$），推导链断裂。因此 lax 幺半的 $f^*$ 下，$f_!$ 的偏差方向**不再由理论保证**，可能表现为 lax（而非 oplax）甚至无确定方向。

> **注**：坍缩传播仅依赖 $f^*$ 保持 $\bot$，不受幺半性类型影响。$f_!$ 的存在性仅依赖 $f^*$ 保持 $\bigwedge$，也不受幺半性类型影响。唯一不兼容 lax $f^*$ 的是涌现偏差 $\delta$ 的方向性。

**结论**：(V2) 要求 oplax 幺半（而非更宽泛的 lax 幺半）是涌现理论（$\delta$ 方向的典范性）的**必要条件**，不可进一步放宽。实际应用中应检查具体的 $f^*$ 是否满足 oplax 方向——对于"大代价被压缩"类型的映射（如 $e^{-\sqrt{c/K}}$, $e^{-c^2/K}$ 等凸对数的情形），oplax 自然成立；对于"小代价被放大"类型的映射（如 sigmoid），则可能仅满足 lax。

### A.3 涌现宽度的可计算性与关联性

**问题**：定义 7.4 给出的涌现宽度 $\mathrm{ew}(f)$ 在多大程度上是可计算或可近似的？

具体而言：
- 对有限值域 $V_l$，$\mathrm{ew}(f)$ 是否总是有限的？
- 计算 $\mathrm{ew}(f)$ 的复杂度是什么？（类比：张量秩的计算是 NP-hard 的（Hillar & Lim, 2013），涌现宽度作为类似的代数分解问题很可能也是。）
- $\mathrm{ew}(f)$ 与经典树宽的严格关系如何？是否存在低树宽但高涌现宽度的实例（反之亦然），从而证明两者度量了正交方向的复杂度？
- 定义中 $g_j$ 的正则性条件（序保持？可计算？连续？）如何选取才能获得良好的数学性质？

**初步观察**：对有限值域 $|V_l| = n$，映射 $\phi : V_l \times V_l \to V_{l'}$ 定义为 $\phi(v,w) = f_!(v \otimes_l w)$ 可视为 $n \times n$ 矩阵（以 $V_{l'}$ 为元素）。涌现宽度即此矩阵在 $(\bigoplus, \otimes_{l'})$-半环上的秩。有限半环上矩阵秩的计算复杂度是已知的困难问题，但对特定半环的结构可能有简化。

### A.4 定理-猜想 (A-opt) 中共享赋值空间兼容性条件的精确化

**问题**：定理-猜想 (A-opt) 要求"共享赋值空间兼容性条件"（A-opt-2），即粗糙层最优赋值集在精细层约束下仍有足够自由度。此条件目前以直觉形式陈述，需要精确的形式化。

**需要严格论证的具体问题**：
- **字典序贪心的正确性条件**：在 $\mathcal{L}$ 为全序时，自顶向下固定赋值的贪心策略需要：粗糙层最优赋值集合 $A_{l_k}^*$ 在精细层的投影 $\rho_{X,S}(A_{l_k}^*)$ 仍包含精细层约束下的（近似）最优解。一个自然的充分条件是**层间变量交互受限**：若层 $l$ 的约束作用域与层 $l'$ 的约束作用域的交集具有有界大小（或有界树宽），则粗糙层最优赋值的"投影自由度"可量化控制。
- **Pareto 前沿的规模控制**：在一般偏序 $\mathcal{L}$ 下，$f_!$ 幺半仅保证评分可逐层独立计算（命题 7.2a），但 Pareto 前沿大小可能为 $\Omega(|A(X)|)$。"可分解"需重新定义为"Pareto 前沿的大小可由各层局部信息界定"或"存在多项式大小的 $\epsilon$-Pareto 近似"。
- **候选充分条件**：(i) 各层约束作用域满足有界树宽；(ii) 层间 $f^*$ 具有"有界扩散"性质（$f^*(v)$ 仅依赖 $v$ 的有限邻域信息）；(iii) $\mathcal{L}$ 的宽度（最大反链大小）有界。这些条件中哪些是必要的、哪些可独立放宽，尚需系统分析。

**结论**：$f_!$ 幺半是优化可分解的**必要但非充分**条件。完整的充分条件需要额外的结构约束（层间变量交互受限或 $\mathcal{L}$ 结构受限），这与经典 CSP 中"有界树宽保证可分解性"的精神一致，但在层间维度上是新的。

### A.5 猜想 7.1(B') 的严格归约构造

**问题**：§7.3 的 3-SAT 归约将所有困难性编码到 $f_!$ 中。强版本 (B') 要求 $f_!$ 的电路复杂度为 $\mathrm{TC}^0$，同时困难性来自非幺半性与约束结构的交互。**关键合法性约束**：在纤维化 CSP 框架中，$f_!$ 不是任意指定的——它必须是某个满足 (V2) 的 $f^*$ 的左伴随。因此，归约构造必须在"伴随合法域"内完成。

**正确的技术路线**：遵循 A.13 的方法一（最严格），**先构造合法的 $f^*$，再由伴随自动导出 $f_!$**，然后验证所得 $f_!$ 的非幺半性和低电路复杂度。

**修订后的构造方向**：考虑 $\mathrm{NAE\text{-}3SAT}$（Not-All-Equal 3-SAT）问题。

1. **纤维选取**：细层 $V_{\text{细}} = ([0,1], \times, \sup)$（概率语义），粗层 $V_{\text{粗}} = ([0,\infty]^{op}, +, \inf)$（代价语义）。

2. **先构造 $f^*$**：取 $f^*(c) = e^{-h(c)}$，其中 $h$ 为凹函数、$h(0)=0$、$h(\infty)=\infty$（如 §A.13 的实例族 $h(c)=(c/K)^\alpha$，$\alpha \in (0,1)$）。由 A.13 的分析，$f^*$ 自动满足 (V2)（保持 $\bot$、保持 $\bigwedge$、oplax 幺半），且 $f_!(p) = h^{-1}(\ln(1/p))$ 由伴随自动导出。

3. **构造约束使全局问题编码 NAE-3SAT**：将 $n$ 个布尔变量嵌入为像素层的 $\{0, 1\}$-取值变量（域 $D_{x_i} = \{0, 1\} \subset [0,1]$），像素层仅有一元约束（独立求解 $O(n)$）。粗层无直接约束。NAE 子句的结构通过约束族的**作用域设计**编码：每个 NAE 子句 $(x_i, x_j, x_k)$ 对应一个三元作用域 $S_{ijk}$，其像素层约束 $\varphi_{S_{ijk}}$ 在三变量取值全同时给出 $\bot_{\text{细}}$。

4. **验证**：(i) $f_!$ 由合法伴随导出，自动在框架合法域内。(ii) $f_!$ 非幺半（由 $f^*$ 严格 oplax 保证，见 §6.2 的完整验证）。(iii) $f_!$ 的电路复杂度：$f_!(p) = h^{-1}(\ln(1/p))$ 对固定 $h$ 是单变量函数，电路复杂度 $O(1)$（$\mathrm{TC}^0$ 内）。(iv) 全局判定"是否存在赋值使所有层非坍缩"等价于 NAE-3SAT 可满足性（NP-complete）。

5. **层级指派与 lax 自然性**：$\Sigma$ 将所有约束作用域指派到细层（$\Sigma(S) = l_{\text{细}}$，除空作用域外），粗层无直接约束。lax 自然性条件在此设定下变为平凡的（粗层约束为空，条件无内容）。

**与旧版构造的区别**：旧版直接手写 $f_!$ 为"子句满足数"并期望其为合法推前函子。但"子句满足数"未必保持 $\bigvee$（joins），从而未必存在满足 (V2) 的右伴随 $f^*$。修订版避免了此问题——$f_!$ 完全由伴随机制导出，合法性由构造保证。困难性不再藏在 $f_!$ 的函数形式中，而是来自约束作用域结构与 $f_!$ 非幺半性的交互——这才是 (B') 试图揭示的真正机制。

**需要完善的**：将以上方向形式化为严格的多项式归约，特别是精确地验证：(i) NAE 子句的编码方式在概率/代价语义下正确捕获 NAE 条件；(ii) 截面化闭包不引入虚假解。

### A.6 与重整化群的严格联系

**问题**：§8.4 指出 $f_!$ 与重整化群中"积分掉快模式"的类比。要将此类比严格化，需要：
- 在连续值域（如 $\mathbb{R}^n$）和统计力学的设定下实例化整个框架。
- 将 $f_!$ 的非幺半性与 Wilson 有效势的非局域性建立精确对应。
- 讨论 $f_!$ 的不动点是否对应于重整化群的不动点（临界现象）。
- 特别地：重整化群中的"不可约顶点函数"（与有效势的非局域部分对应）是否可以用涌现宽度 $\mathrm{ew}$ 来刻画？

### A.7 非偏序层级范畴

**问题**：定义 3.1 要求 $\mathcal{L}$ 为有限偏序集。是否可以自然地推广到更一般的范畴（如含有非平凡自同构的群胚，或含有"平行"态射的范畴）？

在某些应用中，两个层级可能不是严格有序的而是"对等的"——例如自然语言处理中语法和语义可以被视为并行约束系统。这可能需要将 $\mathcal{L}$ 推广到含有非平凡二胞腔的二范畴，但这将显著增加框架的复杂度。

**初步评估**：对含平行态射的 $\mathcal{L}$（如 $l_1 \rightrightarrows l_2$），纤维化结构需处理多条路径的一致性（即 Beck-Chevalley 型条件）。对含群胚结构的 $\mathcal{L}$（如 $l_1 \rightleftarrows l_2$），$f^*$ 与 $g^*$ 互逆，$f_!$ 与 $g_!$ 也互逆，此时坍缩变为对称的（两层级相互决定），涌现消失（$f_!$ 作为等价的一部分自动幺半）。因此群胚推广虽自然但不产生新的涌现现象——真正的涌现依赖于层级的不可逆性。

### A.8 Galois 连接与纤维化语言的表达力权衡

**问题**：如 §3.2 的备注所指出，当所有纤维为完备幺半格（偏序集）时，双纤维化实质退化为**参数化 Galois 连接族**——$f_! \dashv f^*$ 在偏序集之间即 Galois 连接。本框架使用的纤维化机器（截面范畴、lax 自然变换、推前/拉回函子）在多大程度上超出了 Galois 连接的表达力？

**分析**：

1. **Galois 连接直接处理的**：坍缩传播（$f^*$ 保持 $\bot$ 加 Galois 连接的序保持）、$f_!$ 的 oplax 偏差（Galois 连接下 $f_!(a \wedge b) \leq f_!(a) \wedge f_!(b)$ 的幺半版本）、涌现宽度的定义。

2. **纤维化语言提供额外价值的**：
   - **截面范畴** $\Gamma(\mathcal{L}, \mathcal{V})$ 作为整体结构——Galois 连接是逐态射的局部概念，不直接提供"所有层的一致选择"的全局概念。截面正是将局部 Galois 连接粘合为全局对象的工具。
   - **Lax 自然变换**对约束族一致性的统一表述——在 Galois 连接语言中需逐对偏序集手动验证。
   - **函子组合律**自动提供传递性——Galois 连接的组合需要手动验证，而纤维化将其吸收为结构性性质。
   - **推广路径**——从薄纤维到一般纤维的推广在纤维化语言中是无缝的，在 Galois 连接语言中需要完全重建。

**结论**：对于当前框架（薄纤维），Galois 连接语言在局部性质的推导上完全等价，但纤维化语言在全局结构（截面、自然性、组合律）上提供了不可替代的概念简洁性。建议在面向特定应用领域的呈现中使用 Galois 连接的直觉，在理论发展中保留纤维化语言。

### A.9 $f_!$ 的 lax 幺半方向与投影公式

**问题**：命题 4.4 证明了 $f_!$ 典范地为 oplax 幺半（$f_!(v \otimes w) \leq f_!(v) \otimes f_!(w)$）。反方向（lax 幺半）$f_!(v) \otimes_{l'} f_!(w) \leq_{l'} f_!(v \otimes_l w)$ 在何种条件下成立？

**分析**：在代数几何中，**投影公式**（projection formula）$f_!(v \otimes_l f^*(u)) \cong f_!(v) \otimes_{l'} u$ 是一个关键性质，它连接了推前与张量积。若纤维化满足投影公式，则取 $u = f_!(w)$：

$$f_!(v \otimes_l f^* f_!(w)) \cong f_!(v) \otimes_{l'} f_!(w)$$

由伴随 unit $w \leq f^* f_!(w)$ 及 $\otimes$ 保序，$v \otimes_l w \leq v \otimes_l f^* f_!(w)$，从而：

$$f_!(v \otimes_l w) \leq f_!(v \otimes_l f^* f_!(w)) \cong f_!(v) \otimes_{l'} f_!(w)$$

这恢复了 oplax 方向，但**未给出** lax 方向。lax 方向需要 $v \otimes_l f^* f_!(w) \leq v \otimes_l w$ 对适当的条件成立，这等价于 $f^* f_!(w) \leq w$，即伴随的 counit $\varepsilon : f_! f^* \to \mathrm{id}$ 的对偶。但 Galois 连接中 $f_! \dashv f^*$ 的 counit 给出 $f_! f^*(u) \leq u$（不是 $f^* f_!(w) \leq w$），而 unit 给出 $w \leq f^* f_!(w)$——方向相反。

**结论**：lax 方向 $f_!(v) \otimes f_!(w) \leq f_!(v \otimes w)$ 在一般情形下不可从标准条件推出。它成立当且仅当 $f_!$ 是强幺半函子（即 $\delta$ 为等式），此时无涌现。这从反面确认了：**涌现的存在性等价于 lax 方向的失效**。投影公式本身不蕴含也不排除涌现，但为分析涌现的精细结构提供了技术工具。

### A.10 跨纤维的涌现强度比较

**问题**：§4.2 中涌现强度 $\|\delta_f\|_{l'}$ 的定义依赖于目标纤维 $V_{l'}$ 上的度量 $d_{l'}$。当不同的态射 $f : l_1 \to l'$ 和 $g : l_2 \to l'$ 将不同精细层推到同一粗糙层时，$\|\delta_f\|_{l'}$ 和 $\|\delta_g\|_{l'}$ 可以在 $V_{l'}$ 内比较。但对不同目标层的态射 $f : l \to l_1'$ 和 $g : l \to l_2'$，$\|\delta_f\|_{l_1'}$ 与 $\|\delta_g\|_{l_2'}$ 分别位于 $V_{l_1'}$ 和 $V_{l_2'}$ 的度量空间中，不可直接比较。

**可能的归一化方案**：

1. **相对偏差**：定义 $\|\delta_f\|_{\mathrm{rel}} = \|\delta_f\|_{l'} / \mathrm{diam}(V_{l'})$，其中 $\mathrm{diam}$ 为纤维的直径。但无穷值域（如 $[0,\infty]$）的直径为 $\infty$，需要截断或正则化。

2. **范畴内度量**：若纤维化 $p : \mathcal{V} \to \mathcal{L}$ 上存在全局的度量结构（如 $\mathcal{V}$ 本身是度量空间且 $p$ 是短映射），则可在 $\mathcal{V}$ 中直接比较。但这对纤维化施加了很强的额外结构。

3. **序论度量**：利用纤维的格结构定义内禀度量，如 $d_l(v, w) = \mu(v \vee w) - \mu(v \wedge w)$，其中 $\mu$ 是格上的度量函数（模函数）。对布尔格、实数区间等自然纤维，这给出标准度量。但一般完备格不一定有自然的模函数。

**结论**：跨纤维的涌现强度比较本质上需要额外的"跨层校准"结构，这在原始框架中没有也不应该有——因为不同层级的值域代表质性不同的量。涌现宽度 $\mathrm{ew}(f)$ 作为整数值不变量，比度量化的 $\|\delta\|$ 更适合跨纤维比较。

### A.11 层级指派 $\Sigma$ 方向的一般性

**问题**：定义 3.3 要求 $\Sigma : \mathcal{P}(X) \to \mathcal{L}$ 是（协变）函子——更小的作用域映到更精细的层级。此方向性假设在何种应用中失效？如何推广？

**分析**：考虑排课系统。"教室不冲突"约束涉及所有教室的全局赋值（大作用域），但逻辑上属于最强、最基础的约束层——按当前约定应对应最粗糙的层级。而"具体时段偏好"是单个课程的一元约束（小作用域），属于最弱、最精细的层级。此时 $\Sigma$ 的方向与作用域大小一致，框架适用。

然而，在蛋白质折叠中，化学键约束是局部的（涉及少量原子，小作用域）但属于最强的结构层。若将最强层定义为最粗糙层，则小作用域映到粗糙层，$\Sigma$ 必须反转方向。

**可能的推广**：

1. **放弃 $\Sigma$ 的函子性**，将其替换为一般映射 $\Sigma : \{\text{作用域}\} \to |\mathcal{L}|$（仅指派层级，不编码单调性）。代价是丧失定义 3.4 中 lax 自然性的整洁表述——lax 自然性依赖于 $\Sigma$ 诱导的态射来连接不同层级的约束值。

2. **使用反变函子**或更灵活的 **profunctor** $\Sigma : \mathcal{P}(X)^{op} \times \mathcal{L} \to \mathbf{Set}$，编码"作用域 $S$ 的约束可参与到层级 $l$"的关系。这保留了范畴论表述但增加了框架复杂度。

3. **最务实的方案**：将层级指派作为**独立的建模输入**，不与作用域大小绑定，但要求对每个约束 $(S, \varphi_S)$ 单独指定其所属层级 $l_S \in \mathcal{L}$。定义 3.4 的 lax 自然性条件需相应弱化：仅对存在 $\mathcal{L}$ 中态射连接的约束对要求相容性。

**评估**：方案 3 最实用，牺牲最小的理论优雅性。方案 2 最一般但可能过于繁重。本文正文采用原始定义（默认方向），使用者应意识到此约定的适用边界。

### A.12 截面化闭包的修正幅度与"模型病态"判据

**背景**：§3.3 引入的截面化闭包 $\mathrm{cl}$ 解决了旧版本的"筛空"问题：$\mathrm{cl}(\tilde{s})$ 对任意剖面总有定义，且产生合法的 upper-lax 截面。然而，一个相关但不同的问题是：闭包的**修正幅度**是否会过大，以至于截面化评分丧失了原始评分的信息？

**形式化**：对赋值 $a$，定义闭包修正量

$$\Delta_l(a) = \tilde{s}(l) - \mathrm{cl}(\tilde{s})(l) \geq 0$$

（以"$-$"表示在 $V_l$ 中的差，语义因纤维而异）。当所有 $\Delta_l = 0$ 时，闭包无修正；当某些 $\Delta_l$ 很大时，精细层评分被粗糙层结构大幅"拉低"。

**分析**：

1. **$f_!$ 幺半时**：如 A.1 所述，猜测 $\Delta_l(a) = 0$ 对所有非坍缩赋值成立。

2. **$f_!$ 温和非幺半时**：§6.2 的数值实例中 $\Delta_{\text{像素}}(a) = 0.394 - 0.342 = 0.052$（相对修正约 $13\%$），表明在温和的非幺半偏差下闭包修正量可控。

3. **$f_!$ 高度非幺半时**：极端情况下，涌现贡献可能使 $\mathrm{Score}_{l'}$ 急剧恶化，$f^*(\mathrm{Score}_{l'})$ 接近 $\bot_l$，闭包将精细层评分压到接近 $\bot_l$——虽然不是"筛空"（底元素截面是合法截面），但实质上为"闭包性坍缩"。

**模型病态判据**：若存在赋值 $a$ 使得对某层 $l$，$\mathrm{Dir}_l(a) \neq \bot_l$ 但 $\mathrm{cl}(\mathrm{Score}(a))(l) = \bot_l$，则称纤维化实例在 $a$ 处存在**闭包性坍缩**。这意味着层间传播过于激进，可能提示建模问题（如 $f^*$ 的衰减太快、$f_!$ 的涌现贡献过大）。闭包性坍缩是框架对建模者的**诊断信号**，而非框架本身的缺陷。

**与旧版本的对比**：旧版本以"筛选准则"实施 (S4)，风险为筛空（无赋值通过）；新版本以闭包实施 (S4)，底元素截面总存在但可能不具信息量。闭包方案严格优于筛选方案：它同样检测不一致（通过修正量 $\Delta$ 量化），但不会导致框架失效。

### A.13 非幺半 $f_!$ 的系统构造方法

**问题**：如何系统地构造非幺半的 $f_!$ 且保证 $f_! \dashv f^*$ 伴随结构完整？

**方法一（最严格）：选择 oplax 幺半的 $f^*$，左伴随自动非幺半**。

核心思路：$f_!$ 是否幺半完全由 $f^*$ 决定。当 $f^*$ 严格 oplax 幺半（不是强幺半）时，由命题 4.4 知 $f_!$ 为 oplax 幺半，且典型情况下严格——即 $f_!$ 非幺半。

具体构造：对 $f^* : ([0,\infty]^{op}, +) \to ([0,1], \times)$，令 $f^*(c) = e^{-h(c)}$，其中 $h : [0,\infty) \to [0,\infty)$ 递增、$h(0) = 0$、$h(\infty) = \infty$。则：
- $f^*$ 保 $\bot$ 和 $\bigwedge$：自动成立（连续严格递减函数）
- $f^*$ oplax 幺半 $\iff$ $h(c_1+c_2) \leq h(c_1)+h(c_2)$（$h$ 次可加）$\iff$ $h$ **凹**且 $h(0)=0$
- $f^*$ 强幺半 $\iff$ $h(c) = c/K$（线性，即 §6.1 的情形）
- $f_!(p) = h^{-1}(\ln(1/p))$（$h$ 严格递增时 $h^{-1}$ 存在）

**实例族**：取 $h(c) = (c/255)^\alpha$（$\alpha \in (0,1]$），则 $f^*(c) = e^{-(c/255)^\alpha}$，$f_!(p) = 255(\ln(1/p))^{1/\alpha}$。
- $\alpha = 1$：$f^* = e^{-c/255}$（强幺半），$f_! = -255\ln p$（幺半）——§6.1
- $\alpha = 1/2$：$f^* = e^{-\sqrt{c/255}}$（严格 oplax），$f_! = 255(\ln(1/p))^2$（非幺半）——§6.2
- $\alpha \to 0^+$：$f_!$ 的非幺半偏差趋于极端

此实例族以连续参数 $\alpha$ 控制涌现强度：$\alpha$ 从 $1$ 减小时，$f^*$ 的 oplax 偏差增大，$f_!$ 的非幺半偏差增大，涌现效应增强。这提供了一个从"无涌现"到"强涌现"的连续谱。

**方法二：扰动法**。从强幺半的 $f^*_0$ 出发，令 $f^*_\epsilon(c) = e^{-h_\epsilon(c)}$，其中 $h_\epsilon(c) = c/255 + \epsilon \cdot g(c)$（$g$ 为使 $h_\epsilon$ 凹的非线性扰动，$\epsilon$ 为小参数）。此方法保证从幺半到非幺半的连续过渡。

**方法三：统计推前法**（应用导向）。定义 $f_!$ 为包含高阶统计量的函数。此方法在直觉上最自然，但需要额外验证 $f_!$ 是否来自某个 $f^*$ 的伴随。由方法一知：只需检查对应的 $h = -\ln \circ\, f^*$ 是否凹且满足 $h(0)=0, h(\infty)=\infty$。若不满足，则手工构造的 $f_!$ 不在框架的合法范围内——$\delta$ 的方向性不由理论保证。

**评估**：方法一是推荐的标准路径（§6.2 已完整示范）。方法二适用于参数化研究。方法三仅作为启发式，需方法一验证合法性。

**构造模板与可检查条件汇总**。为便于工程使用，将方法一的合法性条件整理为如下检查清单：

| 步骤 | 构造 | 可检查条件 |
|------|------|------------|
| 1. 选择 $h : [0,\infty) \to [0,\infty)$ | 递增函数，$h(0)=0$，$h(\infty)=\infty$ | 直接验证 |
| 2. 定义 $f^*(c) = e^{-h(c)}$ | 自动满足保 $\bot$ 和保 $\bigwedge$ | 无需额外验证 |
| 3. 验证 oplax 幺半 | $h$ 次可加：$h(c_1+c_2) \leq h(c_1)+h(c_2)$ | 等价于 $h$ **凹**且 $h(0)=0$ |
| 4. 导出 $f_!(p) = h^{-1}(\ln(1/p))$ | 由伴随自动确定 | 无需手工选择 |
| 5. 判定涌现 | $f_!$ 非幺半 $\iff$ $h$ 严格凹（非线性） | 等价于 $h \neq c/K$（非仿射） |

当 $h(c) = (c/K)^\alpha$（$\alpha \in (0,1]$）时，$\alpha = 1$ 无涌现，$\alpha < 1$ 有涌现，$\alpha$ 越小涌现越强。此实例族以单参数 $\alpha$ 连续控制涌现强度，适合作为理论验证与参数化实验的标准测试平台。
