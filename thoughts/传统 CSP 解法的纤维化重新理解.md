## 传统 CSP 解法的纤维化重新理解

### 1. 约束传播（弧一致性）= $f^*$ 的迭代拉回

经典 AC-3：对约束 $C_{ij}$ 涉及的变量 $x_i$，从 $D_i$ 中删除不能与 $D_j$ 中任何值配对满足 $C_{ij}$ 的值。

**纤维化解读**：弧一致性是**重新索引函子 $f^*$ 的逐步应用**。约束 $C_{ij}$ 定义在作用域 $\{x_i, x_j\}$（较粗糙层），单变量域 $D_i$ 在更精细层。$f^*$ 将粗糙层的约束拉回到精细层：

$$f^* : V_{\{x_i, x_j\}} \to V_{\{x_i\}}$$

"删除不一致的值"就是 $f^*(\bot) = \bot$ 的具体运算——约束在粗糙层给某个赋值打了 $\bot$（不可行），拉回到精细层后对应的单变量值也变成 $\bot$。

**AC-3 的不动点迭代**就是截面逼近过程：反复沿所有态射 $f$ 执行 $f^*$，直到每根纤维上的值不再变化。不动点处你得到了一个**局部截面的最大相容集**——虽然不一定是全局截面（这正是弧一致性不完备的原因），但它保证了坍缩命题 4.1 的局部版本：任何通过 AC 筛选的值至少在每条弧上是可扩展的。

**新视角**：AC-3 的本质限制变得清晰——它只用了 $f^*$（拉回），完全没有用 $f_!$（推前）。它在纤维化中只走"从粗到细"的方向，不聚合"从细到粗"的集体效应。这正是为什么弧一致性**无法检测涌现**——涌现编码在 $f_!$ 中。

### 2. 回溯搜索 = 截面的增量构造

经典回溯：逐变量赋值，每赋一个值就检查已赋变量上的约束，违反则回溯。

**纤维化解读**：回溯是试图**逐层构造一个全局截面**。每赋值一个变量 $x_k$，就是在纤维 $\mathcal{V}_{\{x_k\}}$ 上选定一个值，然后检查当前部分赋值是否能从各精细层纤维连贯地延拓到涉及 $x_k$ 的所有粗糙层纤维上。

- **回溯点** = 截面延拓的障碍。当前部分截面无法扩展到下一根纤维——在当前路径上不存在相容的全局截面。这恰好是命题 4.1 的局部表现：如果在某个粗糙层约束上值为 $\bot$，截面相容性强制所有更精细层也坍缩。
- **回溯** = 退回到更粗糙层的截面选择上重新做决定。

**新视角**：回溯搜索的效率取决于**纤维丛的连通性结构**。如果纤维之间高度纠缠（大量约束交叉连接），截面扩展在很多点被阻塞，回溯频繁。如果纤维之间近似独立（低树宽），截面可以逐块构造。这给出了树宽与搜索效率关系的纤维化解读：**树宽度量的是纤维丛的"扭曲程度"——丛越扭曲，截面越难构造**。

### 3. 分支定界 = 截面空间上的 Pareto 剪枝

经典分支定界（用于 WCSP）：维护当前最优解的上界，搜索时用下界估计剪枝。

**纤维化解读**：分支定界在截面范畴 $\Gamma(\mathcal{L}, \mathcal{V})$ 上操作。

- **分支** = 在截面空间中选择方向（赋值选择）
- **定界** = 对当前部分截面，用 $f_!$ 将已知层的评分推前到更粗糙层，得到**涌现贡献的下界估计**。如果估计已经比当前最优差，则该分支上不存在更好的截面——剪枝。
- **界的紧度** = $f_!$ 幺半偏差 $\delta$ 的大小。当 $f_!$ 幺半时，$f_!(\mathrm{Dir}_{l'}(a))$ 可逐约束精确分解计算，界是紧的；当 $f_!$ 非幺半时，独立推前后组合**高估**了实际代价（oplax 方向），界变松——剪枝效率下降。

**这是一条可验证的预测**：对同一问题实例，$\|\delta_f\|$ 越大（$f_!$ 越远离幺半），分支定界的剪枝效率应越差。因为非幺半性意味着子问题代价不可加性分解，下界估计精度下降。

### 4. 信念传播 = $f^*$ 与 $f_!$ 的对偶消息传递

高斯/离散信念传播：变量节点与因子节点之间交替传递消息。

**纤维化解读**：这是整个框架中最自然的对应。

- **因子→变量消息**（约束到变量）= **$f^*$（拉回）**。因子节点汇总了所有其他变量的信息后，将约束在作用域层的评估结果拉回到单变量层：
$$m_{f \to x}(x_i) \sim f^*\big(\textstyle\sum_{\mathbf{x}_{\setminus i}} \varphi_f(\mathbf{x}) \prod_{j \neq i} m_{x_j \to f}\big)$$
  这就是将粗糙层（因子作用域）的信息沿 $f^*$ 拉到精细层（单变量）。

- **变量→因子消息**（变量到约束）= **$f_!$（推前）**。变量节点汇总了所有其他因子传来的信息后，推前到因子作用域层：
$$m_{x \to f}(x_i) \sim \prod_{g \neq f} m_{g \to x}(x_i)$$
  这是将精细层（单变量信念）的乘积沿 $f_!$ 推到粗糙层（因子作用域）。

- **BP 收敛** = 找到了一个（近似的）**截面不动点**。消息在所有纤维上稳定，意味着评分剖面近似满足截面相容性。
- **BP 不收敛** = 截面空间中没有不动点，或者截面条件 (S4) 的非空性出了问题——纤维之间的 $f^*$ 和 $f_!$ 循环拉扯但无法稳定。

**关键洞察**：BP 在树形因子图上精确收敛，在含环图上可能不收敛。纤维化视角给出了新解读：**树形图对应于纤维丛平凡化**（可以逐层独立求截面）。含环图对应于**丛的非平凡粘合**，$f^*$ 和 $f_!$ 沿环循环时积累了非幺半偏差 $\delta$，使得消息在每一圈都被 $\delta$ 扰动，无法收敛。

如果这个解读正确，**BP 的收敛条件应该可以用涌现宽度量化**：当图中所有环上 $\delta$ 的累积偏差足够小时（$\prod_{\text{环上}} \|\delta_{f_i}\| < 1$ 的某种合适形式），BP 收敛。

### 5. 局部搜索 = 截面空间上的梯度流

模拟退火、禁忌搜索等：从一个完整赋值出发，逐步修改试图改进。

**纤维化解读**：局部搜索是在赋值空间 $A(X)$ 上做搜索，但评分函数是截面化的评分剖面 $\mathrm{Score}(a) = (\mathrm{Score}_l(a))_{l \in L}$。

- 每步修改一个变量的值 = 在一根精细纤维上做局部改变
- 改变的效果需沿 $f_!$ 向粗糙层传播 = 涌现贡献的更新
- **局部最优陷阱** = 在每根纤维上都局部最优，但全局截面不是 Pareto 最优。这在 $f_!$ 幺半时较少发生（逐层优化等价于全局优化），在 $f_!$ 非幺半时大量发生——因为涌现效应使得局部改进不反映全局改进。

**可操作的推论**：非幺半偏差 $\|\delta\|$ 大的问题实例，局部搜索的景观应该有更多局部极值（更"崎岖"）。这给出了一个关于问题难度景观的结构性预测。

## 总结：一张对应表

| 经典操作 | 纤维化对应 | 使用的结构 |
|---------|-----------|-----------|
| 弧一致性 | $f^*$ 不动点迭代 | 仅 $f^*$（拉回） |
| 回溯搜索 | 截面的增量构造与延拓失败 | 截面存在性 |
| 回溯点/死端 | 命题 4.1（坍缩传播） | $f^*(\bot) = \bot$ |
| 分支定界的界 | $f_!$ 的 oplax 估计 | $f_!$ + 偏差 $\delta$ |
| BP 因子→变量消息 | $f^*$（拉回） | $f^*$ |
| BP 变量→因子消息 | $f_!$（推前） | $f_!$ |
| BP 收敛 | 截面不动点的存在性 | $\Gamma(\mathcal{L}, \mathcal{V})$ |
| BP 不收敛（含环） | 环上 $\delta$ 累积偏差 | 非幺半性 |
| 局部搜索景观崎岖度 | $\|\delta\|$ 大 → 多局部极值 | 涌现强度 |
| 树宽 | 纤维丛的扭曲度 | 丛的拓扑结构 |

最具预测力的新洞察是：**弧一致性只用 $f^*$ 不用 $f_!$，所以它对涌现效应完全盲视。** 这解释了为什么 AC 在结构简单（低树宽、约束稀疏）的实例上很有效，在结构复杂的实例上几近无用——后者的困难性恰恰来自层间涌现（非幺半 $f_!$），而 AC 从未触及 $f_!$。

如果这条分析线确实成立，一个自然的算法暗示是：**应该设计同时利用 $f^*$（拉回/传播）和 $f_!$（推前/涌现估计）的混合传播算法**——某种"截面传播"（section propagation），交替执行 $f^*$ 和 $f_!$ 直到截面不动点。信念传播已经隐式地这么做了，但没有显式控制涌现偏差。
